System Requirements

BBEdit 11.5 requires OS X 10.9.5 or later, and is fully compatible with OS X 10.11 "El Capitan". It will not run on any earlier version of Mac OS X.

Feature Highlights

BBEdit 11.5 offers many powerful features for editing and processing text and code, and for managing your work. Here are some highlights:

  • New single-window Find Differences results!
  • Enhanced clipping management for greater flexibility
  • Greatly expanded syntax coloring support (both current and as a basis for future expansion)
  • A selection of carefully crafted pre-installed color schemes
  • Info popup offers live document statistics and file permissions adjustment
  • New editing commands, including keyboard-based rectangular selection
  • Optionally move BBEdit's app support folder to iCloud Drive

as well as all the powerful core features BBEdit is known for, including:

  • Support for high-resolution Retina displays
  • Go menu, with Function, Jump Points, and Markers navigation
  • Configurable, per-language Preview Filters
  • Project document-based website configurations (with support for basic content deployment)
  • Automatically preserves unsaved documents upon quit
  • Save and load custom text color schemes
  • Enhanced navigation in project windows and editing windows
  • Simplified preferences and configuration management
  • Direct editing of files within Zip archives—plus multi-file search & replace!
  • Text completion for easy insertion of words, syntax elements, and clippings
  • Preview in BBEdit supports template pages and stylesheets
  • In-window Live Search to highlight and quickly jump between matches
  • Modeless Find and Multi-File Search windows provide a consistent, familiar interface to BBEdit's legendary search and replace capabilities
  • Package support for convenient extensibility
  • Persistent global Unix worksheet for quick command line chores
  • Streamlined Script and Text Filter management

Info on New Features

In addition to these major enhancements, BBEdit 11.5 also contains numerous additional behavioral and interface refinements, as well as performance enhancements and bug fixes. For a detailed summary of changes and bug fixes, please refer to the current release notes, which are available on our web site.
