Customize the export settings

The Advanced Export option allows you to customize the export settings such as file format, frame rate, audio and video quality, etc.

Customize the export settings:

  1. Select Share > Advanced Export.
  2. Enter a file name in the Export As field and select a location.
  3. Select the file format from the Export menu. Click the Options button to select additional video and audio options.
  4. Click Export.
