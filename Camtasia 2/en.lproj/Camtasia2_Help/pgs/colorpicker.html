Working with the color picker

The color picker allows you to select a color by hue, saturation, and brightness. It is accessible via the properties panel for annotations and select visual effects. Use RGB values, Hex codes, or the eyedropper tool to match colors from images or other media.

To select a color:

  1. Click the color circle to open the color picker.


  2. Click and drag along the slider to select a hue.


  3. Click and drag within the palette to fine-tune the color.

Revert to System Color Picker:

The System Color Picker is a standard feature of the Mac OS. You can opt to use it instead of the built-in color picker.

  1. Select Camtasia 2 > Preferences > General tab.
  2. Check Enable System Color Picker.

Customize color swatches:

Color swatches can be customized, providing easy access to frequently used colors.

  1. Click the color circle to open the colors dialog.
  2. Select the swatch you want to change.

  3. Select a new color.

Use RGB or Hex values:

  1. Click the color circle to open the colors dialog. The RGB or Hex values are located at the bottom.


  2. Select the appropriate field and input a new value.


    To switch between RGB and Hex values, click the arrow.

Use the eyedropper tool:

  1. Click the color circle to open the colors dialog.
  2. Click the eyedropper icon. The eyedropper turns blue and a magnifying glass appears.


  3. Hover the magnifying glass over the color you want to match.
  4. Click to select the color.
