Change the default duration

You can change the default duration for images, shapes, text, transitions, and animations. The default duration for images, shapes, and text is five seconds. The default duration for transitions and animations is one second.

Change the default duration:

  1. Select Camtasia > Preferences > Timeline tab.
  2. Enter a time in seconds into the duration field.

    Note: Changing the default duration does not change the duration for existing media on the timeline. See Change the duration of assets on the timeline.
