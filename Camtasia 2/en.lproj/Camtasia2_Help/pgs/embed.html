Embed a video on a web page or blog

MP4 videos are ideal for sharing your video content on a web browser or if your audience will view the content on different operating systems. You can create a video (MP4) in Camtasia to embed on a web page or blog.

Export as web page:

  1. Select Share > Export as Web Page.
  2. Enter a file name in the Export As field and select a location.
  3. Drag the slider to adjust file size versus video quality.
  4. Click Export.

Embed a video (MP4) on a web page or blog:

  1. Select Share > Export as Web Page. Name the file and click Export.
  2. Open the source code for the index.html file by completing any of the following:
  • Open the HTML file in an HTML editor, such as Adobe Dreamweaver.
  • Control-click the HTML file and select Open With > TextEdit.
  • Double-click the HTML file. In a web browser, select the View Source option. In Safari, select View > View Source.
