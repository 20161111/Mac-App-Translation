Import media from Google Drive

Import videos, images, or audio from Google Drive into a project to include as source files for a video.

  1. Select File > Import From Google Drive.
  2. The Google Account dialog appears. Enter your email and password, then click Sign in. If you do not have a Google account, you will need to create one to continue.
  3. Read through the terms of service and click Accept.
  4. The Import From Google Drive dialog appears. Click to select a video, image, or audio file to import into Camtasia. Command-click to select multiple files.
  5. Click Import.

The files are added to the media bin.
