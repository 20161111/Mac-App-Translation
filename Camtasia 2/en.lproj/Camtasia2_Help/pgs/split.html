Split

Split divides one clip into two parts. You can move and edit each clip separately. To split, select one of these options from the Edit menu:

  • Split selected at playhead (Command-T): Select this option to split the currently selected clip at the location of the playhead.
  • Split all (ripple split) at playhead (Command-Shift-T): Select this option to split all clips at the location of the playhead.
  • Ripple split and shift at playhead (Hold Command and drag the playhead): Use this option to insert space at the location of the playhead.
