Add a table of contents

For videos shared to Screencast.com or exported as a web page, you can add a table of contents based on the markers on the timeline. The table of contents adds navigation points to allow viewers to jump to specific sections within your video.

Add a table of contents:

  1. Select View > Show Marker Track.
  2. The marker track appears.
    • To add a global marker, click within the markers track in the area above the playhead. Or, press Option-M.

    • To add a media-based marker, click above the media to add a marker.

All markers added to the timeline appear as table of contents entries. You cannot select which markers to display in the table of contents.

  1. Enter a marker name. The marker names appear as the table of contents entry.
  2. Table of contents is only available when sharing to Screencast.com or exporting as a web page. Select one of the following to export a video with a table of contents:
    • For Screencast.com: Go to Share > Screencast.com. Log in and select the "Create table of contents from markers" option. Click Share.
    • For export: Go to Share > Export as Web Page and select the "Create table of contents from markers" option. Click Export.

Search content in video:

Within the table of contents, the search field allows viewers to search for caption text, slide titles, table of contents entries, and markers.

  1. Click the Table of Contents button on the player bar.
  2. Enter the search term into the field.

  3. Select a result. The video jumps to the location the text appears within the video.
