Export an MP4 video

MP4 videos are ideal for sharing your video content on a web browser or if your audience will view the content on different operating systems.

Export an MP4 video:

  1. Select Share > Export.
  2. Enter a file name in the Export As field and select a location.
  3. Click Export.

Export as web page:

  1. Select Share > Export as Web Page.
  2. Enter a file name in the Export As field and select a location.
  3. Drag the slider to adjust file size versus video quality.
  4. Click Export.
