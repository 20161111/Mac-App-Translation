Export captions

Export a SRT or SAMI caption file to translate the caption text to another language and to share captions in another program or with others. Camtasia exports the caption text and time data. See Caption file formats to learn more about these file formats.

Export captions:

  1. Select Share > Captions.
  2. Enter a file name and select the format from the dropdown.
  3. Click Save.
