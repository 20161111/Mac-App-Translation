Save or extend a single frame

Extend the frame at the end of a clip:

You can extend a single frame of video on the timeline. Use this option to extend the video frame to match the duration of the audio or voice narration.

  1. Click to select a clip on the timeline.
  2. Drag the playhead to the position you want to extend the last frame of the video to.
  3. Select Edit > Playhead > Extend Frame to Playhead.

Save a frame as an image:

You can save a single frame of the timeline as a still PNG image at the playhead.

  1. Drag the playhead on the timeline to the desired frame.
  2. Select File > Save Frame As.
  3. Enter a filename and select a location. Click Save.

    To use the still image in the current project, select File > Import Media.
