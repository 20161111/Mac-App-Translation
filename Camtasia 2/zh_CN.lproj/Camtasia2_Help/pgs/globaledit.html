Global edit mode

Use global edit mode to change all of the keyframes on selected media at one time.

When to use global edit mode:

  1. When editing the properties of an animated annotation, use global edit mode to ignore the animation.

    For example, if an arrow is animated to move across the canvas and then you decide to change the color, border, or size of the annotation, enable global edit mode before editing to make an absolute change to the arrow properties before, during, and after the animation.
  2. When copying an animated annotation for reuse in a project, use global edit mode to make a relative change to the basic properties (position, rotation, etc.).

    For example, if one arrow is animated to point to one item and move down a list, and you want to use that same animation but change the position of the arrow on the canvas, enable global edit mode before changing the beginning state (pre-animation). The positioning at the end state will then be relative to the start point instead of reverting to the positioning of the original arrow.
