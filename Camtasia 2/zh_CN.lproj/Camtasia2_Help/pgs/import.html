Import media from your computer

Import recordings, video clips, audio clips, or images from your computer into a project to include as source files for a video.

Import media from your computer:

  1. Select File > Import Media or Press Command-I.
  2. Select the file(s) to import. Hold Command and click to select multiple files.
  3. Click Import.

    The files are added to the media bin in Camtasia.
