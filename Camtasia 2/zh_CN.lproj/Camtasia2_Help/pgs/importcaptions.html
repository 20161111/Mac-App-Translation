Import captions

Camtasia supports importing SAMI and SRT file formats. See Caption file formats to learn more about these file formats.

Import captions:

  1. Select File > Import Media.
  2. The import dialog appears. Select a supported caption file format (SAMI or SRT) and click Import.
  3. The caption file is added to the timeline.
