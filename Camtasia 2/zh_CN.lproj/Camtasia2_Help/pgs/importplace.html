Import from My Places

Import your files from a shortcut in My Places, such as OneDrive for Business.

Import from My Places

  1. In Camtasia, select File > Import From My Places.
  2. Select a location from My Places.

    If you do not have any locations in My Places or to import from a new location, see Add or remove a Place.
  3. Browse to select the file to import.
  4. Click Import.

The file opens in the media bin.
