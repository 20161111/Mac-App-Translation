Open a Camtasia project

A Camtasia project file (.cmproj) saves a copy of the recordings, source files, and edits completed within a video project. To share Camtasia for Mac projects across computers, save the Camtasia project file as a standalone project.

To open a Camtasia project file:

  • Double-click a Camtasia project icon.
  • Open Camtasia and select File > Open Project or Open Recent.
