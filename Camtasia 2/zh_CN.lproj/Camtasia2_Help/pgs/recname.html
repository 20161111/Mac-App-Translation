Change the recording name

Camtasia provides a default recording name. You can provide a name for your recording while the recording is paused or you can change the recording name in the Media tab after recording.

Name the recording while recording:

  1. While recording, press Command-Shift-2 or select the menu bar icon > Pause Recording.
  2. Enter a recording name in the Name field and click Complete to end the recording or Resume to continue recording. After recording, the recording file appears in the Media tab with the new name.

Change the recording name in the Media tab:

  1. Control-click on the recording in the Media tab and select Rename.
  2. Enter a new name and press Enter.
