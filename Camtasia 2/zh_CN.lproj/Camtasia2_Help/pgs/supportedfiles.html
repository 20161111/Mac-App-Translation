Supported file types

You can import the following file types into a Camtasia for Mac project:

  • MOV
  • MP4
  • AIFF
  • MP3
  • WAV
  • AAC (M4A)
  • JPEG
  • PNG
  • TIFF
  • PDF (single page only)
  • SRT or SAMI (caption file formats)

The following file types are unsupported:

  • AVI
  • WMV
  • WMA
  • DV
  • MTS
  • MKV
  • MPEG-1
  • MPEG-2
  • AVCHD
  • RM (RealMedia)
  • SWF
  • FLV
  • M4P (protected AAC)
  • 3GP
  • OGG
  • OGV
  • DGP
  • EPS
  • AI
  • APP
  • SVG
  • WebM
  • PSD

For unsupported files, try converting the files to another format. Display file information using MediaInfo and do conversions with MPEGStreamclip, VLC, Handbrake, or Quicktime Pro to a format that Camtasia for Mac will read. We recommend QuickTime or MP4/H.264 (AVC) formats for importing. Audio files can be converted with iTunes or audio editors such as Audacity, Sound Studio, or Amadeus Pro.
