Timeline tracks

Tracks represent the sequence of all media within a video. Drag media and assets onto a track on the timeline to include it in the video.

The track order controls how media is displayed within the video. The top track displays media front-most in the video.

Change the track height:

To adjust the track height, click and drag the border between two tracks on the left side of the timeline.

Camtasia remembers the track height when you save the project.

Remove a track:

You can create an unlimited amount of tracks. You may want to remove empty tracks.

To remove a single track:

  1. Control-click to the left of the track you want to remove.
  2. Select Remove Track from the context menu. If there is any media on the track, a message will appear asking if you want to delete the track and all media on the track.

To remove all empty tracks:

  1. Control-click to the left of a track.
  2. Select Remove All Empty Tracks from the context menu.

Insert a track:

You can create an unlimited amount of tracks. You may want to insert a new track while editing.

  1. Control-click to the left of a track.
  2. Select Insert Track from the context menu.
