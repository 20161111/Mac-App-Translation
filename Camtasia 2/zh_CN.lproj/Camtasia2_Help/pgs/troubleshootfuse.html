Troubleshooting TechSmith Fuse

If TechSmith Fuse cannot connect or send files to Camtasia, see the following topics for more information:

  • Unable to send files from Fuse to Camtasia
  • Unable to scan the QR code
  • QR code is missing
  • Specified port is already in use

For additional help, see the support article on the TechSmith website.

Unable to send files from Fuse to Camtasia:

If you are unable to send files from TechSmith Fuse to Camtasia, make sure:

  • Your mobile device and the computer running Camtasia are on and connected to the same network.
  • Camtasia is open on the computer when files are sent from Fuse.
  • Mobile Sharing is turned on. Select Camtasia 2 > Preferences > General tab to check.
  • You have scanned the QR code in the Connect Mobile Device dialog to establish a connection between your mobile device and Camtasia. See Import media from a mobile device for more information.

Unable to scan the QR code:

To send files from your mobile device to Camtasia, you must scan the QR code using TechSmith Fuse. In Camtasia, select File > Connect Mobile Device to open the QR code. If Fuse cannot scan the QR code, try the following:

  • Make sure your mobile device is connected to the same Wi-Fi network as the computer running Camtasia.
  • Aim the camera on the mobile device at the QR code on the computer screen.
  • Position the camera so that the QR code is centered within the corners of the Scanner screen in Fuse.

QR code is missing:

In Camtasia, an error message appears in the Connect Mobile Device dialog (File > Connect Mobile Device) if a QR code cannot be generated:

A QR code may not be generated if:

  • The network/router is disconnected or needs to be reset.
  • The specified port is already in use.

If a QR code is not generated:

  • Check your network or firewall settings.
  • Connect your mobile device and computer to another network.

Specified port is already in use:

Mobile Sharing must be on for Camtasia to receive files from TechSmith Fuse. If the specified port is in use by another application, Mobile Sharing will not turn on. To manually change the port number:

  1. Select Camtasia 2 > Preferences > General tab.
  2. Enter a port number that is not in use by another application.
    Note: Mobile Sharing must be turned off to change the port number.

  3. Click Turn On Sharing.
