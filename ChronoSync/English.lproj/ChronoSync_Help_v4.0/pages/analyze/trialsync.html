Trial Synchronizations

For peace of mind, run trial synchronizations before synchronizing. Trial synchronizations analyze all the files but don't actually synchronize. Instead, a list of file changes are displayed so you can see what happens before it happens.

If you select a file-pair in the window, you can view information on the pair by clicking on the "File/Folder Information" disclosure triangle. With the information provided, you can see basic information for both files as well as the differences (denoted in red). When a file-pair is selected, certain actions can be performed upon these files. These actions are listed below.

In the center of the Trial Synchronization window, statistics about the files are listed. Specifically it shows the number of conflicts, deletions and the which direction the files will be synchronized. There is also a summary of the number of bytes that will be copied as part of this synchronization.

Available actions in the Trial Synchronization window and their key equivalents:

Compare - This function simply opens the two files in their respective applications. This allows you to compare their contents to each other and is useful for determining which file is the correct one when a conflict arises. You can change what applications are used to compare two files by defining a "Helper Application" in the "Preferences". Key Equivalent: CMD-I.

Exclude - ChronoSync will ignore a file if it is excluded. Entire folder branches can be marked for exclusion. This command is saved with the ChronoSync document. This can be undone by using the "Reset" action command. Key Equivalent: CMD-Down Arrow.

Force Synchronize - Overrides the file’s default status and forces it to be synchronized in the specified direction on every synchronization. This can be undone by using the "Reset" action command. Key Equivalent: OPTION-SHIFT-CMD-Right Arrow (or Left Arrow if forcing left).

Force Synchronize (Once) - Overrides the file’s default status and forces it to be synchronized in the specified direction for the next synchronization only. It is automatically reset once the synchronization is performed. This can be undone by using the "Reset" action command. Key Equivalent: OPTION-CMD-Right Arrow (or Left Arrow if forcing left).

Delete - Deletes a file in one target or deletes both files. Will ask you to choose, if it is not obvious, which file should be deleted. When multiple files are selected, and it isn't obvious, a list appears with each selected file. Key Equivalent: CMD-K.

Reset - This option is available after a force synchronized or delete option was chosen. It will reset the file-pair status to its original state. Key Equivalent: CMD-U.

Toggle Checkbox - Will toggle the state of the item's checkbox. Key Equivalent: SPACEBAR.

Show Info in Finder - Opens a 'Get Info' window showing file information for the selected file or folder.

Reveal in Finder - Opens a Finder window to where the selected file or folder resides.

Open - If a folder is selected, a Finder window will open, showing the contents of the folder. If a file is selected, ChronoSync will attempt to open the file. If ChronoSync is unable, an error message will occur.

Note: Clicking on the checkbox column header will toggle the state of all the checkboxes in the list based on the state of the first entry (i.e. if first entry is OFF, all entries become ON and vice versa).
