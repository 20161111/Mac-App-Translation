Document Organizer

The Document Organizer window is a convenient way to keep track of and manage all your synchronizer and container documents. Basic file functionality such as renaming, duplicating and deleting can be done right from the Document Organizer window. The options within the window make opening and scheduling documents a simple task.

The Document Organizer is just an easier way to access documents while using ChronoSync.
