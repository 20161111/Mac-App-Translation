Document Organizer

Document List Actions

When a document is selected in the list, actions can be performed upon that document by right-clicking (or ctrl-clicking). Doing this will bring up a contextual menu. Here's an explanation of each action:



  • Opens the selected document. When opening a document, ChronoSync will make an attempt to connect to any remote targets. You may also open the selected document by clicking the "Open" button at the bottom of the list.

  • Opens the selected document and runs a synchronization immediately. If opening a container document, all contained documents will get synchronized.

  • Opens the selected documents and performs a trial synchronization immediately on it. Note that this action is not available for container documents.

  • Opens the "Add Scheduled Synchronization" window. Here you can set up a schedule for the selected document to run. For help on scheduling, browse the Scheduling section. Note that if you already have a schedule for this particular document, you are NOT editing the schedule but instead you are adding a new schedule for this document. If this is the case, once saved, you will then have multiple schedules for that document. You may also schedule the selected document by clicking the "Schedule..." button at the bottom of the list.

  • Invokes the "Rename Document" sheet where you can specify a new filename for the selected document. Note that any schedule that references this document will be updated accordingly. This action can also be executed by selecting "Rename..." in the "File" menu.

  • Invokes the "Duplicate Document" sheet where you can specify a filename for the duplicate. If no filename is given, " copy" will be appended to the original filename. This action can also be executed by selecting "Duplicate..." in the "File" menu.

  • Invokes the "Remove or Delete Item" sheet. Here you can choose whether you want to remove the reference to the document (i.e, remove it from the document list) or permanently delete the document (i.e, will be deleted from your hard drive). This action can also be executed by selecting "Delete..." in the "File" menu.

  • Reveals the selected document's location in Finder.

  • Opens a 'Get Info' Finder window for the selected document.

  • When you hover over an item in the list, an 'Info' button will appear to the right of your selection. Clicking it brings up a window with much more detailed information about the selected document.

It is good practice to perform file changes in the Document Organizer rather than via Finder. If, for example, you wish to permanently delete a document and you do so in Finder, once ChronoSync determines the document isn't available, it will dim the entry in the documents list. ChonoSync will not remove the entry from the list because it's not sure if it's been deleted or if it's just on a volume that's currently unavailable. If you truly want it out of the documents list, you'll have to perform a "Remove..." on it manually.

To open a document that is not currently available in the list, choose "Open Other..." to invoke a file selection dialog for browsing to the desired file.
