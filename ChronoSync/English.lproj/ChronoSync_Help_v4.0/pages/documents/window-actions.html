Document Organizer

Document Window Actions

To the right of the Document Organizer window, there are actions that can be performed that are not related to the documents in the list. Here is an explanation of these actions:

  • Opens a new, untitled synchronizer document. A sheet drops down so that you can specify a filename, if you desire. Note that the sheet only appears if you have the "Maintain documents folder:" preference setting enabled. The folder that is chosen is where the new document will be kept.

  • Opens a new, untitled container document. A sheet drops down so that you can specify a filename, if you desire. Note that the sheet only appears if you have the "Maintain documents folder:" preference setting enabled. The folder that is chosen is where the new document will be kept.

  • Opens the ChronoSync Help Center so that you may search for specific topics or learn more about using ChronoSync.

  • Opens a browser window and directs you to our Support Ticket section on our website.

  • When enabled, the Document Organizer window will automatically show/hide as you open documents or switch applications.
