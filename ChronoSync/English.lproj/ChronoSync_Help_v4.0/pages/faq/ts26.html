ChronoSync FAQ's

I am getting the "Error: Date roll-back encountered at" error message. What is this?

A date roll back means that the file has changed since the last sync but now has an older modification date. Since new files should have a newer date, it does not replace the file but gives you an error. It is recommended you try and determine why this is happening first before copying over any files. Take a look at our "Date Roll-back" tech note to get more detailed information.
