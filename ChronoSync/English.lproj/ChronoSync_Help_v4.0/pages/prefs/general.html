Preferences

General Preferences

There are several settings in the "General" preferences that can be used to set general ChronoSync defaults. Each available setting is described below.

Application Behavior:

Launch at login - Launches ChronoSync once login is complete. Hide will hide ChronoSync after it's launched at login.

Show ChronoSync menu in menubar - Determines whether the ChronoSync menu will be displayed in the menubar. The "Flash menu for notices" option, when enabled, will flash an 'X' in the menubar icon when there is an error in the scheduled synchronization and a exclamation point will flash when an update is available. The "Animate during scheduled syncs" option will animate the menubar icon when a scheduled sync is running.

Automatically check for updates - Determines if ChronoSync will automatically check for newer versions.

Keep backup copy of documents - Maintains a backup copy of all Synchronizer and Container documents. The backup copy has a tilde (~) appended to its name. This way, if a Synchronizer document were to somehow get corrupted during a save operation, a backup copy exists from which custom document settings can be salvaged.

Maintain documents folder - Determines whether or not to maintain a dedicated folder for ChronoSync documents. You can "Choose..." the destination. Enabling this setting dictates whether you will be greeted by a naming sheet when creating new documents.

Window Behavior:

Default Size - Defines the default document size that new documents will open with. The "Current" button will set the default size to the current document's size.

Remember size & position - Indicates whether or not the size and position of windows is saved with documents.

Interface animations - Indicates whether or not various user interface animations should be displayed.

Refresh archive outline on activate - Expanded entries in the "Archive" panel will refresh when switching from another application to ChronoSync.

'Resume' document windows - Previously opened document windows will automatically open if enabled. This setting is only available on Mac OS X 10.7 ("Lion") or later.

Always show Document Organizer on launch - When enabled, will display the Document Organizer window on every launch except when running a scheduled synchronization.

Auto-show/hide the Document Organizer - Will automatically show/hide as you open documents or switch applications.

Compact toolbars - The document editor and log windows will use a compact-style toolbar. This is only available for Mac OS X 10.10 and later.

Synchronization Behavior:

Play sound on success - Plays a sound when a synchronization successfully completes.

Play sound on failure - Plays a sound when a synchronization fails to complete without errors.

Play attention sound - Plays a sound when the user's attention is required.

Send emails via - The method in which email notifications are sent. ChronoSync defaults to sending email via Apple Mail. Aside from Apple Mail, there are other popular clients that are supported. ChronoSync can also send email directly without using an email client at all.

Note: Sounds can be overridden on a per document basis in the Sounds Option panel.
