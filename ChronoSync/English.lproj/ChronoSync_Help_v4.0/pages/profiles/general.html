Connection Profiles

Connections -- General

In the General panel, you can define a connection profile so that you can connect to a remote or local agent or to an InterConneX sharespace. To connect to an agent remotely, you must have ChronoAgent installed on the remote machine, the one you wish to access. To connect to an InterConneX sharespace, you must have InterConneX installed on your desired device..

Setting up a connection profile:

  1. Provide the necessary information to set up the connection profile. Each field is explained below.
  2. Test the connection to the agent or sharespace by clicking on the "Test..." button.
  3. Once everything is set up correctly and you are happy with the settings, you can choose "Next" to go to Advanced settings.

General panel options explained:

Profile Name - Give your connection profile a name. Connection profile names must be unique.

Connect to - Choose a remote agent or sharespace from the list that you wish to connect to. You may connect by choosing an agent or sharespace that you already set up or by using an internet address to access an agent. If you choose the "Internet Address" option, you will have to provide an IP address (or a DNS Name) and a port number. It is mandatory that the port number you choose must be the same as the one you chose when you installed and configured the ChronoAgent you are trying to connect to.

Login As - Choose to login as a "Guest" or as a authenticated user.

Username/Password - If not logging is as "Guest", provide a username and password to gain access to the agent or sharespace.

Test... - Attempts to make a brief connection based on the supplied settings, reporting success or failure depending on the results.
