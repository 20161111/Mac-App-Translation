Adding Rules

The "Intermediate" and "Advanced" rules can be a bit involved. There are no predefined rules. Instead you must explicitly add rules to the rule base. There is a great amount of flexibility with both of these modes. In the "Intermediate" mode, rules are added to an inclusion list. An inclusion list contains rules that a file must pass to be considered for synchronization.

In the "Advanced" mode, rules are added to both an inclusion and an exclusion list. As mentioned above, the inclusion list contains rules that a file must pass to be considered for synchronization. The exclusion list, on the other hand, list rules that, if a file passes, excludes them from consideration.

To add a rule to an inclusion or exclusion list:

  1. Click on the icon.
  2. From the drop down sheet, choose "Create your own rule of the following type:" and then the type of rule you wish to add.
  1. Select "OK".
  2. The rule's setup sheet will appear where you can now input the information you want for that rule.
  3. Select "OK".
  4. You will now see that the rule appears in the "Rules" panel list.
