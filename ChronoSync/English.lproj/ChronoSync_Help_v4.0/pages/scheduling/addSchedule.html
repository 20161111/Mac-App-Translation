Scheduling

Adding a Schedule

The scheduler can be used to automatically set up the synchronization of Synchronizer and/or Container documents. You can add as many documents as you would like to the "Scheduled Documents Manager". You can even schedule the same Synchronizer document several times. They do not have to be open for the synchronization to occur.

Adding current document to scheduler:

  1. With the desired document open, choose "Schedule..." in the "Actions" menu. This will create a new scheduled synchronization. A Synchronizer document can be scheduled multiple times. Note: The document must be saved before you can schedule it.
  2. After the "Add Schedule" sheet appears, provide the necessary information to specify the interval.
  3. If you are satisfied with this schedule, then choose "OK".
  4. The entry will now appear in the "Scheduled Documents Manager" window.

Note: The scheduler may also be invoked by selecting the "Schedule..." option in the toolbar.

Adding a document to the scheduler via the Document Organizer

  1. Open the "Document Organizer" window by choosing "Document Organizer" in the "Window" menu.
  2. Look in the document list and select the document you want to schedule. Note: If there are no documents in the list, you can add them by choosing "Open Other..." and navigating to the document.
  3. Once the document is selected, choose "Schedule...".
  4. The "Add Schedule" sheet appears. Provide the necessary information to specify the interval.
  5. If you are satisfied with this schedule, then choose "OK".
  6. The entry will now appear in the "Scheduled Documents Manager" window.

Adding a non-open document to scheduler

  1. Open the "Scheduled Documents Manager" window by choosing "Scheduled Documents Manager" in the "Window" menu. As a shortcut, it may also be opened by choosing the "Open Scheduler" toolbar option.
  2. As a default, the window appears in the top right corner of your screen. To add an entry to it, click on the button on the window.
  3. After the "Add Schedule" sheet appears, choose the "Choose..." button to navigate to the document you wish to schedule.
  4. Provide the necessary information to specify the interval.
  5. If you are satisfied with this schedule, then choose "OK".
  6. The entry will now appear in the "Scheduled Documents Manager" window.

Scheduling Options:

Only run if target volumes/agents are available - This item will be scheduled to run at the designated time but if the target volumes or agents are not presently available, it will not run. If this option is left OFF and the targets aren’t available, an error will be generated.

Prompt user before running - With this option ON, when a scheduled sync comes due, the user is prompted via the Notification Center as to whether or not the sync should be run. The prompt will remain visible until the user answers it, causing all subsequent syncs for the same item to be ignored until answered.

Run no more than once every... - When a document is scheduled to be run, ChronoSync will check the timestamp on the previously run synchronization. If the set value falls within the elapsed time, the synchronization will not run again.

Abort if still running at... - When you turn this ON, a time of day must be specified. If the scheduled sync is still running when that time of day is reached, it is suspended. This is intended to halt any long running syncs so they don’t interfere with daily activity, such as an overnight sync that may occasionally take a very long time to complete.

Retry on errors - If any errors occur during the synchronization, ChronoSync will reschedule the sync to run. Any scheduled item that shows "w/error" next to its last run time would qualify. "Retry on errors" takes precedence over "Run no more than once every...". This is new behavior for version 4.5.

Interval between retries - The number of minutes to wait before running the scheduled item again after an error. This time delay begins once the synchronization finishes.

Maximum retry attempts - The maximum number of retry attempts before giving up.
