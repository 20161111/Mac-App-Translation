Synchronizer Documents

Synchronizer documents contain all of the information necessary to keep two targets synchronized with each other. This includes a reference to each target, the type of synchronization to be performed, numerous option settings to control ChronoSyncâ€™s behavior and zero or more rules to inform ChronoSync of what type of files need to be synchronized. Just as with any other document-based application, Synchronizer documents may be saved to disk.

A new "Untitled" Synchronizer document can be created by either:

  • Choosing "New" from the "File" menu and then selecting “Synchronizer”.
  • Choosing "Create a new synchronizer document" in the Document Organizer window.
  • Using the shortcut keys, COMMAND-N.

If you have the "Maintain documents folder:" preference setting enabled, when you open a new, untitled synchronizer document, a sheet drops down so that you can specify a filename, if you desire. The folder that is chosen is where the new document will be kept.

If "Always show Document Organizer on launch" is checked in General Preferences, you can "Create a new synchronizer document" by selecting that button in the Document Organizer window.

Once you create and save a Synchronizer document, you always run your synchronizations or backups from the same document.
