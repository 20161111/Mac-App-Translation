Analyze Panel Actions

Once a set of files is selected, various actions can be performed upon them. These actions can be invoked through the "Actions" menu or you can customize your toolbar to include the action commands on it. Some commands, including the "Exclude" and "Force Synchronize" commands, are saved with the ChronoSync document and used every time a synchronization is run.

Available actions in the Analyze panel:

Compare - This function simply opens the two files in their respective applications. This allows you to compare their contents to each other and is useful for determining which file is the correct one when a conflict arises. You can change what applications are used to compare two files by defining a "Helper Application" in the "Preferences".

Restore - Restores using the previously copied version of the file with the purpose of recovering lost or damaged files. For example, if a file is designated for a Left-to-right synchronization, restore will replace the file on the left side with the file on the right side. In the case of bidirectional syncs, you can restore the file on either target.

Exclude - ChronoSync will ignore a file if it is excluded. Entire folder branches can be marked for exclusion. This command is saved with the ChronoSync document.

Include - Includes a file or folder that has been previously excluded.

Force Synchronize - Overrides the file’s default status and forces it to be synchronized in the specified direction on every synchronization. This command is saved with the ChronoSync document. This can be undone by using the "Reset" action command.

Force Synchronize (Once) - Overrides the file’s default status and forces it to be synchronized in the specified direction for the next synchronization only. It is automatically reset once the synchronization is performed. This command is saved with the ChronoSync document. This can be undone by using the "Reset" action command.

Delete From Left/Right... - Deletes a file from the target specified. You can "Delete Immediately", "Move to Trash", or, when an Archive is available, "Move to Archive".

Reset Direction - This option is available after a force synchronized option was chosen. It will reset the file-pair status so that a forced synchronization will not occur.

Synchronize Selection - Only the selected file-pair(s) will be immediately synchronized.

Trial Synchronize Selection... - Only the selected file-pair(s) will be immediately trial synchronized. The trial synchronize sheet will then appear.

Refresh File List - The state of the files displayed in the outline view will be re-evaluated based on the actual state of the files as stored on disk. This normally happens automatically, but sometimes ChronoSync needs to be told by the user when a filesystem change occurs.

Show Info in Finder - Opens two 'Get Info' windows, showing file information for both the left and right target file.

Reveal in Finder - Opens two Finder windows to where the selected file or folder resides.

Open - If a folder-pair is selected, two Finder windows will open, showing the contents of each folder. If a file-pair is selected, ChronoSync will attempt to open the files. If ChronoSync is unable, an error message will occur.

Note: All of these actions may be performed on multiple files. Multiple files may be selected by holding down either the SHIFT key or COMMAND key when selecting with your mouse. The SHIFT key will select a range (contiguous) of files/folders, whereas the COMMAND key will select noncontiguous cells.
