Analyze Panel Restore

The Analyze Panel Restore function performs an "opposite sync" whereby files are copied in the opposite direction of a standard synchronization with the purpose of recovering lost or damaged files. In the case of bidirectional syncs, you can restore the file on either target.

Analyze Restore options explained:

Restore only items synchronized by this document - When checked, only items synced by that particular Synchronizer document will be restored. If unchecked, all files that exist on the selected target will be restored to the destination, even if they were not placed there by ChronoSync. Note that with this option turned on, all rules and exclusions that exist in the document are pertinent and thus rule/explicit excluded files will not be restored. With this setting OFF, however, rules and exclusions have no effect on the restore process.

Replace existing items - If checked, existing files will be replaced. In the case of trying to recover accidentally deleted files, you would NOT want to have this setting turned ON. With the setting OFF, Restore will only copy back a file if it has no counterpart. This is a safe, quick and convenient way to recover files when you might not know the scope of you data loss an you just want to recover what's missing.

only if replacing with...version - Replaced files can be limited to only older or newer files.
