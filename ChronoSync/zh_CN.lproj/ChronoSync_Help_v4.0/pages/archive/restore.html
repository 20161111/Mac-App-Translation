Restoring Archived Items

Restoring is used to revert back to an older version of a file, which may have been either accidentally or intentionally deleted or overwritten. When restoring, you can replace the file on either target or you may also restore to a specific folder on any hard drive. The latter would be used when you want to work with an older version of a file but you don't necessarily want that to become the current version of a file.

Archive Restore options explained:

Restore File to Left Target - The selected archived file will be restored to the left target. Will place the file in question in the appropriate location within the left target.

Restore File to Right Target - The selected archived file will be restored to the right target. Will place the file in question in the appropriate location within the right target.

Specific Folder - A file browse window will appear so that a specific location can be selected for the archived file to be restored to.
