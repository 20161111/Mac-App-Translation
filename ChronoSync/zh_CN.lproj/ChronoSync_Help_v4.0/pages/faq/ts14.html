ChronoSync FAQ's

Can I create a bootable backup with ChronoSync?

Yes. You can create a bootable backup with ChronoSync. Choose "Bootable Left-to-right" or "Bootable Right-to-left" in the "Operation" menu on the Setup panel to make a bootable copy of your system. When you select this, ChronoSync asks for your user password to ensure this operation is being performed by a user with administrative privileges. Once this is determined, the document changes its settings to reflect the default settings for bootable backups.
