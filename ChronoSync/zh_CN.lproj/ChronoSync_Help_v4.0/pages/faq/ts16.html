ChronoSync FAQ's

I need an older version of a document. How can I restore this file?

Each time a file is modified, a version of that file will be moved to the Archive folder (assuming "Archive replaced files" is enabled in the "Setup" panel). That way, if an old file ever needs to be used again, you can view it and access it in the "Archive" panel.

This is done by using the "Restore" operation. Simply choose the file(s) you wish to restore, then choose "Restore" from the "Actions" menu. You will then choose the location to which you wish to restore this file.
