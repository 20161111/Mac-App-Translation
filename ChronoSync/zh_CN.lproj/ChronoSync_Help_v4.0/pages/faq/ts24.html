ChronoSync FAQ's

Can I synchronize my two Photos libraries into one?

ChronoSync can keep Photos or any Apple program on two Macs in sync but it can not merge data together. What's the difference? ChronoSync can replace the newer data files with the older data files but not merge two files together. Why? Because most of the iApps or Apple programs store their data in a unique way that can only be created by importing data using the application.
