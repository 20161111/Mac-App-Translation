ChronoSync FAQ's

What is ChronoAgent?

ChronoSync has the ability to synchronize targets by communicating with "agents". An agent is simply a faceless program that runs external to ChronoSync. It can be installed on the same machine as ChronoSync or on another computer and accessed via a network connection. ChronoAgent is installed via a separate installer. It can be installed on any Mac running Mac OS 10.5 or greater. The ChronoAgent, or remote agent, gives ChronoSync full root access to destination Macs.

The advantages of synchronizing via agents are numerous:

  • Full administrative access to all files regardless of the privileges of the logged in user who is running ChronoSync.
  • The ability to communicate with remote machines.
  • WAN access settings to connect to an agent over the internet.
  • Users don't need to be locally logged into remote machines.
  • User/group ID mapping between machines.
  • Limitations of an AFP connection are circumvented.
  • In the future, agents can be written that run on non-Mac OS X based systems.
