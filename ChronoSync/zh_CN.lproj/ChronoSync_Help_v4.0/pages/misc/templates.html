Document Templates

As you’ve probably noticed, ChronoSync documents contain numerous settings and options that will control your synchronizations. Since you may find yourself always enabling and disabling the same set of options for each Synchronizer document you create, you may want to create default settings that ChronoSync uses when creating new documents. Making use of Templates allows you to do just that.

Templates are simply Synchronizer or Container documents that are used as starting points for new documents that you create. If you set up a document and find that the settings you set are common ones throughout all your documents, it may be a good idea to make this a template.

It is important to note that any rules that were set in the document will also be saved in a template.

To create a Template:

  1. With the desired document open and all settings set up as you wish, choose the "Save As Template..." option in the "File" menu. This gives you the ability to save the settings of the current document as a template file.
  2. A sheet will drop down for you to specify a template name.
  3. If you are satisfied with the template name, then choose "OK".
  4. The new template will now appear, by display name, in the "New" submenu under the "File" menu.

Template management can be performed in the "Templates" Preference pane. Here you can remove previously created or specify other Synchronizer documents that should be added as templates.
