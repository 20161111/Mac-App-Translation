Options

The options available in ChronoSync help you with customizing your sync scenario. Below, you'll see each group of options listed for both Synchronizer documents and Container documents.

Synchronizer Document Options

  • Reporting & Error Handling
  • Special File/Folder Handling
  • Archive Handling
  • Sync Triggers
  • Notifications
  • Scripts
  • Sounds

Container Document Options

Container documents have their own set of options. Only two - "Disable user interaction for errors & warnings" and "Display summary after synchronization completes" - will override options chosen within the individual Synchronizer documents. These options are very similar to those found in a Synchronizer document. Only the "Reporting & Error Handling" options vary.

  • Reporting & Error Handling
  • Notifications
  • Scripts
  • Sounds
