Options

Scripts

In the "Scripts" section, you can set up scripts to be executed either before or after the synchronization runs. ChronoSync can execute AppleScript files, shell script files and applications as a pre- or post- sync 'script'. In the case of applications, ChronoSync will simply launch the application but will not wait for it to terminate. ChronoSync will patiently wait for the scripts to finish executing before continuing. Check the appropriate box based on when you want a script to run and then select the “Choose...” button to select the script that you would like to run.

Scripts options explained:

Pre-Synchronization Script - Will run a script prior to the start of synchronization.

Abort on script errors - If errors are encountered while running the script, the synchronizer execution will abort.

Post-Synchronization Script - Will run a script after a synchronization has completed.

Skip on synchronization errors - If a synchronization error occurred, the selected “post” script will not run.

Each script has an associated "Run" button. This allows the scripts to be executed without the need to run the Synchronizer document.

Note: AppleScripts must be saved as "Script", not as an application or other format.
