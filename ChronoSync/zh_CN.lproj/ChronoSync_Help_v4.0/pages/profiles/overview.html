Connection Profiles

Connections Overview

Connection profiles can be defined so that you can connect to remote agents or InterConneX sharespaces. To connect to an agent remotely, you must have ChronoAgent installed on the remote machine, the one you wish to access. To connect to an InterConneX sharespace, you must have InterConneX installed on your desired device. The Connections Preferences panel lists all of the profiles that have been previously set up.

To add a new connection profile, click on the add button. When you do this, the General panel is displayed so that you can provide the necessary information to set up the connection profile.

Note: At the bottom of each panel, a wizard is there to guide you.
