Editing Rules

To edit a rule while in the "Simple" rule-base mode is as easy as enabling the rule and editing its contents, then supplying it with the necessary criteria.

In the "Intermediate" or "Advanced" rule-base mode, rules in either the inclusion or exclusion list may be easily edited by following the steps below.

To edit a rule from an inclusion or exclusion list:

  1. Select the rule that you wish to edit by highlighting it in the list.
  2. Click on the icon.
  3. The rule will present itself in a drop down sheet for you to edit it.
  4. Click "OK" when finished and you will see the changes in the list.
