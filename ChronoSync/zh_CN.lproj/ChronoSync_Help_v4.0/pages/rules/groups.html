Rule Groups

Rule groups act much like parenthesis do in a mathematical expression. A rule group is evaluated as a whole and the result is applied to the next rule or rule group in the expression. The entire rule group must evaluate to TRUE - and any other rules that may be set - for the file to be included or excluded.

Some advantages of using rule groups are as follows:

  • Keep certain rules grouped together.
  • Contain multiple rules evaluated by AND and OR relationships.
  • Allow a set of multiple rules to be saved as a single rule template.
  • Can be nested as deeply as you like.

Follow these steps to add a Rule Group:

  1. While in "Intermediate" or "Advanced" rule mode, select located above the list.
  2. Name your rule group.
  3. Select "OK" and you'll then see it appear in the list.
  4. Rules can be added by highlighting the rule group and adding a new rule or by dragging previously defined rules into the group.
