Scheduling

The scheduler in ChronoSync can be used to automatically set up the synchronization of documents to run at any time of the day. You may schedule as many documents for synchronization as you wish. The scheduler is able to perform the synchronization, regardless of whether or not the document is already open. ChronoSync automatically installs a separate program call the ChronoSync Scheduler, which quietly and silently monitors the scheduled items database. When a scheduled synchronization is due to occur, the ChronoSync Scheduler will launch ChronoSync, instruct it to do its job, then quit ChronoSync, all without requiring user intervention.
