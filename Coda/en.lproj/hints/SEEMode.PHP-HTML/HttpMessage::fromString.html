HttpMessage::fromString

Example

 static public HttpMessage HttpMessage::fromString ([ string $raw_message [, string $class_name = "HttpMessage" ]] )

Description

Create an HttpMessage object from a string. This function alias is deprecated and only exists for backwards compatibility reasons. The use of this function is not recommended, as it may be removed from PHP in the future.

Return Values

Returns an HttpMessage object on success or NULL on failure.
