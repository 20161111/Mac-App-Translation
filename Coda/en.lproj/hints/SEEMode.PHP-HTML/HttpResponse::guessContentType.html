HttpResponse::guessContentType

Example

 static string HttpResponse::guessContentType ( string $magic_file [, int $magic_mode = MAGIC_MIME ] )

Description

Attempts to guess the content type of supplied payload through libmagic. If the attempt is successful, the guessed Content-Type will automatically be set as response Content-Type.

Return Values

Returns the guessed content type on success or FALSE on failure.
