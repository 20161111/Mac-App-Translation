com_event_sink

Example

 bool com_event_sink ( variant $comobject , object $sinkobject [, mixed $sinkinterface ] )

Description

Instructs COM to sink events generated by comobject into the PHP object sinkobject. Be careful how you use this feature; if you are doing something similar to the example below, then it doesn't really make sense to run it in a web server context.

Return Values

Returns TRUE on success or FALSE on failure.
