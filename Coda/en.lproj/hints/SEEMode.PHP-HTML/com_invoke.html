com_invoke

Example

 mixed com_invoke ( resource $com_object , string $function_name [, mixed $function_parameters ] )

Description

com_invoke() invokes the method named function_name of the COM component referenced by com_object. com_invoke() returns FALSE on error, returns the function_name's return value on success. All the extra parameters function_parameters are passed to the method function_name. Example #1 Don't use com_invoke(), use OO syntax instead
<?php// do this$val = $obj->method($one, $two);// instead of this:$val = com_invoke($obj, 'method', $one, $two);?>
