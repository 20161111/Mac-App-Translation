fseek

Example

 int fseek ( resource $handle , int $offset [, int $whence = SEEK_SET ] )

Description

Sets the file position indicator for the file referenced by handle. The new position, measured in bytes from the beginning of the file, is obtained by adding offset to the position specified by whence. In general, it is allowed to seek past the end-of-file; if data is then written, reads in any unwritten region between the end-of-file and the sought position will yield bytes with value 0. However, certain streams may not support this behavior, especially when they have an underlying fixed size storage.

Return Values

Upon success, returns 0; otherwise, returns -1.
