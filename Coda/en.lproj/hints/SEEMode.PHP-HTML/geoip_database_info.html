geoip_database_info

Example

 string geoip_database_info ([ int $database = GEOIP_COUNTRY_EDITION ] )

Description

The geoip_database_info() function returns the corresponding GeoIP Database version as it is defined inside the binary file. If this function is called without arguments, it returns the version of the GeoIP Free Country Edition.

Return Values

Returns the corresponding database version, or NULL on error.
