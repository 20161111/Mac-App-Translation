geoip_isp_by_name

Example

 string geoip_isp_by_name ( string $hostname )

Description

The geoip_isp_by_name() function will return the name of the Internet Service Provider (ISP) that an IP is assigned to. This function is currently only available to users who have bought a commercial GeoIP ISP Edition. A warning will be issued if the proper database cannot be located.

Return Values

Returns the ISP name on success, or FALSE if the address cannot be found in the database.
