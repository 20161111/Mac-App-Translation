gettimeofday

Example

 mixed gettimeofday ([ bool $return_float = false ] )

Description

This is an interface to gettimeofday(2). It returns an associative array containing the data returned from the system call.

Return Values

By default an array is returned. If return_float is set, then a float is returned. Array keys: "sec" - seconds since the Unix Epoch "usec" - microseconds "minuteswest" - minutes west of Greenwich "dsttime" - type of dst correction
