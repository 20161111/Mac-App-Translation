gzdeflate

Example

 string gzdeflate ( string $data [, int $level = -1 ] )

Description

This function compress the given string using the DEFLATE data format. For details on the DEFLATE compression algorithm see the document "» DEFLATE Compressed Data Format Specification version 1.3" (RFC 1951).

Return Values

The deflated string or FALSE if an error occurred.
