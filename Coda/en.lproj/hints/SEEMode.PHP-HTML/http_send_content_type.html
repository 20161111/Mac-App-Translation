http_send_content_type

Example

 bool http_send_content_type ([ string $content_type = "application/x-octetstream" ] )

Description

Send the Content-Type of the sent entity. Note: This function should be used in conjunction with http_send_data(), http_send_file() and http_send_stream().

Return Values

Returns TRUE on success or FALSE on failure.
