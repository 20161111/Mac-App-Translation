http_send_file

Example

 bool http_send_file ( string $file )

Description

Sends a file with support for (multiple) range requests. This functions behaviour and further action is dependent on the following INI settings: http.send.not_found_404 and http.log.not_found. If the INI setting http.send.not_found_404 is enabled and the INI setting http.log.not_found points to a writable file, a log message is written when the file was not found.

Return Values

Returns TRUE on success or FALSE on failure.
