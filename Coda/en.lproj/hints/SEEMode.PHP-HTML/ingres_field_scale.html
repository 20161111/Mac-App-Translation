ingres_field_scale

Example

 int ingres_field_scale ( resource $result , int $index )

Description

ingres_field_scale() returns the scale of a field. This value is used only for the decimal SQL data type. For detailed information, see the Ingres OpenAPI User Guide, Appendix "Data Types" in the Ingres documentation. Note: Related Configurations See ingres.array_index_start in Runtime Configuration

Return Values

Returns the scale of the field, as an integer
