isset

Example

 bool isset ( mixed $var [, mixed $... ] )

Description

Determine if a variable is set and is not NULL. If a variable has been unset with unset(), it will no longer be set. isset() will return FALSE if testing a variable that has been set to NULL. Also note that a NULL byte ("\0") is not equivalent to the PHP NULL constant. If multiple parameters are supplied then isset() will return TRUE only if all of the parameters are set. Evaluation goes from left to right and stops as soon as an unset variable is encountered.

Return Values

Returns TRUE if var exists and has value other than NULL, FALSE otherwise.
