ldap_first_attribute

Example

 string ldap_first_attribute ( resource $link_identifier , resource $result_entry_identifier )

Description

Gets the first attribute in the given entry. Remaining attributes are retrieved by calling ldap_next_attribute() successively. Similar to reading entries, attributes are also read one by one from a particular entry.

Return Values

Returns the first attribute in the entry on success and FALSE on error.
