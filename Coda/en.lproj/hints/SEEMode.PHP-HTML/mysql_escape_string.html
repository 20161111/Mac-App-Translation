mysql_escape_string

Example

 string mysql_escape_string ( string $unescaped_string )

Description

This function will escape the unescaped_string, so that it is safe to place it in a mysql_query(). This function is deprecated. This function is identical to mysql_real_escape_string() except that mysql_real_escape_string() takes a connection handler and escapes the string according to the current character set. mysql_escape_string() does not take a connection argument and does not respect the current charset setting. Warning: This function has been DEPRECATED as of PHP 5.3.0. Relying on this feature is highly discouraged.

Return Values

Returns the escaped string.
