mysql_list_fields

Example

 resource mysql_list_fields ( string $database_name , string $table_name [, resource $link_identifier ] )

Description

Retrieves information about the given table name. This function is deprecated. It is preferable to use mysql_query() to issue an SQL SHOW COLUMNS FROM table [LIKE 'name'] statement instead.

Return Values

A result pointer resource on success, or FALSE on failure. The returned result can be used with mysql_field_flags(), mysql_field_len(), mysql_field_name() and mysql_field_type().
