mysql_ping

Example

 bool mysql_ping ([ resource $link_identifier ] )

Description

Checks whether or not the connection to the server is working. If it has gone down, an automatic reconnection is attempted. This function can be used by scripts that remain idle for a long while, to check whether or not the server has closed the connection and reconnect if necessary. Note: Since MySQL 5.0.13, automatic reconnection feature is disabled.

Return Values

Returns TRUE if the connection to the server MySQL server is working, otherwise FALSE.
