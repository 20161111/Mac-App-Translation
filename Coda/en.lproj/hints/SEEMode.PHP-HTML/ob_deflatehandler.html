ob_deflatehandler

Example

 string ob_deflatehandler ( string $data , int $mode )

Description

Deflate output handler

Return Values

For use with ob_start(). Note: This output handler can only be used once. The deflate output buffer handler can only be used once. It conflicts with ob_gzhandler() and zlib.output_compression as well and should not be used after mbstring extension's mb_output_handler() and session extension's URL-Rewriter (AKA session.use_trans_sid).
