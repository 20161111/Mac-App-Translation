odbc_primarykeys

Example

 resource odbc_primarykeys ( resource $connection_id , string $qualifier , string $owner , string $table )

Description

Returns a result identifier that can be used to fetch the column names that comprise the primary key for a table.

Return Values

Returns an ODBC result identifier or FALSE on failure. The result set has the following columns: TABLE_QUALIFIER TABLE_OWNER TABLE_NAME COLUMN_NAME KEY_SEQ PK_NAME
