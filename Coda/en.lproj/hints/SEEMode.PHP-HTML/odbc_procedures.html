odbc_procedures

Example

 resource odbc_procedures ( resource $connection_id )

Description

Lists all procedures in the requested range.

Return Values

Returns an ODBC result identifier containing the information or FALSE on failure. The result set has the following columns: PROCEDURE_QUALIFIER PROCEDURE_OWNER PROCEDURE_NAME NUM_INPUT_PARAMS NUM_OUTPUT_PARAMS NUM_RESULT_SETS REMARKS PROCEDURE_TYPE
