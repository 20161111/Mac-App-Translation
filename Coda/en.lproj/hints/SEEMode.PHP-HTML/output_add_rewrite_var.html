output_add_rewrite_var

Example

 bool output_add_rewrite_var ( string $name , string $value )

Description

This function adds another name/value pair to the URL rewrite mechanism. The name and value will be added to URLs (as GET parameter) and forms (as hidden input fields) the same way as the session ID when transparent URL rewriting is enabled with session.use_trans_sid. Please note that absolute URLs (http://example.com/..) aren't rewritten. This function's behavior is controlled by the url_rewriter.tags php.ini parameter. Note: Calling this function will implicitly start output buffering if it is not active already.

Return Values

Returns TRUE on success or FALSE on failure.
