pg_escape_string

Example

 string pg_escape_string ([ resource $connection ], string $data )

Description

pg_escape_string() escapes a string for querying the database. It returns an escaped string in the PostgreSQL format without quotes. pg_escape_literal() is more preferred way to escape SQL parameters for PostgreSQL. addslashes() must not be used with PostgreSQL. If the type of the column is bytea, pg_escape_bytea() must be used instead. pg_escape_identifier() must be used to escape identifiers (e.g. table names, field names) Note: This function requires PostgreSQL 7.2 or later.

Return Values

A string containing the escaped data.
