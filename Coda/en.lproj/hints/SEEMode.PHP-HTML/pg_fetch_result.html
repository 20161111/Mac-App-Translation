pg_fetch_result

Example

 string pg_fetch_result ( resource $result , int $row , mixed $field )

Description

string pg_fetch_result ( resource $result , mixed $field )
pg_fetch_result() returns the value of a particular row and field (column) in a PostgreSQL result resource. Note: This function used to be called pg_result().

Return Values

Boolean is returned as "t" or "f". All other types, including arrays are returned as strings formatted in the same default PostgreSQL manner that you would see in the psql program. Database NULL values are returned as NULL. FALSE is returned if row exceeds the number of rows in the set, or on any other error.
