pg_get_result

Example

 resource pg_get_result ([ resource $connection ] )

Description

pg_get_result() gets the result resource from an asynchronous query executed by pg_send_query(), pg_send_query_params() or pg_send_execute(). pg_send_query() and the other asynchronous query functions can send multiple queries to a PostgreSQL server and pg_get_result() is used to get each query's results, one by one.

Return Values

The result resource, or FALSE if no more results are available.
