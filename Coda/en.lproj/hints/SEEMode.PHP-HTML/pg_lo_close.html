pg_lo_close

Example

 bool pg_lo_close ( resource $large_object )

Description

pg_lo_close() closes a large object. large_object is a resource for the large object from pg_lo_open(). To use the large object interface, it is necessary to enclose it within a transaction block. Note: This function used to be called pg_loclose().

Return Values

Returns TRUE on success or FALSE on failure.
