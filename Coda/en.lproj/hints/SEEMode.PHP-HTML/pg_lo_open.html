pg_lo_open

Example

 resource pg_lo_open ( resource $connection , int $oid , string $mode )

Description

pg_lo_open() opens a large object in the database and returns large object resource so that it can be manipulated. Warning Do not close the database connection before closing the large object resource. To use the large object interface, it is necessary to enclose it within a transaction block. Note: This function used to be called pg_loopen().

Return Values

A large object resource or FALSE on error.
