pg_lo_read

Example

 string pg_lo_read ( resource $large_object [, int $len = 8192 ] )

Description

pg_lo_read() reads at most len bytes from a large object and returns it as a string. To use the large object interface, it is necessary to enclose it within a transaction block. Note: This function used to be called pg_loread().

Return Values

A string containing len bytes from the large object, or FALSE on error.
