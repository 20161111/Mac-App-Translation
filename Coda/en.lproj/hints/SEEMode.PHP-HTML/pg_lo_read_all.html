pg_lo_read_all

Example

 int pg_lo_read_all ( resource $large_object )

Description

pg_lo_read_all() reads a large object and passes it straight through to the browser after sending all pending headers. Mainly intended for sending binary data like images or sound. To use the large object interface, it is necessary to enclose it within a transaction block. Note: This function used to be called pg_loreadall().

Return Values

Number of bytes read or FALSE on error.
