pg_meta_data

Example

 array pg_meta_data ( resource $connection , string $table_name )

Description

pg_meta_data() returns table definition for table_name as an array. Warning: This function is EXPERIMENTAL. The behaviour of this function, its name, and surrounding documentation may change without notice in a future release of PHP. This function should be used at your own risk.

Return Values

An array of the table definition, or FALSE on error.
