readline_callback_handler_install

Example

 bool readline_callback_handler_install ( string $prompt , callback $callback )

Description

Sets up a readline callback interface then prints prompt and immediately returns. Calling this function twice without removing the previous callback interface will automatically and conveniently overwrite the old interface. The callback feature is useful when combined with stream_select() as it allows interleaving of IO and user input, unlike readline().

Return Values

Returns TRUE on success or FALSE on failure.
