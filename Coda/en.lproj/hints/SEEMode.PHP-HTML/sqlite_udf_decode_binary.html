sqlite_udf_decode_binary

Example

 string sqlite_udf_decode_binary ( string $data )

Description

Decodes binary data passed as parameters to a UDF. You must call this function on parameters passed to your UDF if you need them to handle binary data, as the binary encoding employed by PHP will obscure the content and of the parameter in its natural, non-coded form. PHP does not perform this encode/decode operation automatically as it would severely impact performance if it did.

Return Values

The decoded string.
