ssh2_sftp_lstat

Example

 array ssh2_sftp_lstat ( resource $sftp , string $path )

Description

Stats a symbolic link on the remote filesystem without following the link. This function is similar to using the lstat() function with the ssh2.sftp:// wrapper in PHP 5 and returns the same values.

Return Values

See the documentation for stat() for details on the values which may be returned.
