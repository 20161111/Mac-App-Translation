strripos

Example

 int strripos ( string $haystack , string $needle [, int $offset = 0 ] )

Description

Find position of last occurrence of a string in a string. Unlike strrpos(), strripos() is case-insensitive.

Return Values

Returns the numerical position of the last occurrence of needle. Also note that string positions start at 0, and not 1. If needle is not found, FALSE is returned. Warning: This function may return Boolean FALSE, but may also return a non-Boolean value which evaluates to FALSE, such as 0 or "". Please read the section on Booleans for more information. Use the === operator for testing the return value of this function.
