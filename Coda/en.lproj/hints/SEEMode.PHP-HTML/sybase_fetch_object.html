sybase_fetch_object

Example

 object sybase_fetch_object ( resource $result [, mixed $object ] )

Description

sybase_fetch_object() is similar to sybase_fetch_assoc(), with one difference - an object is returned, instead of an array. Speed-wise, the function is identical to sybase_fetch_array(), and almost as quick as sybase_fetch_row() (the difference is insignificant).

Return Values

Returns an object with properties that correspond to the fetched row, or FALSE if there are no more rows.
