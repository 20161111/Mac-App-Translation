sybase_fetch_row

Example

 array sybase_fetch_row ( resource $result )

Description

sybase_fetch_row() fetches one row of data from the result associated with the specified result identifier. Subsequent call to sybase_fetch_row() would return the next row in the result set, or FALSE if there are no more rows.

Return Values

Returns an array that corresponds to the fetched row, or FALSE if there are no more rows. Each result column is stored in an array offset, starting at offset 0. Data types PHP Sybase
