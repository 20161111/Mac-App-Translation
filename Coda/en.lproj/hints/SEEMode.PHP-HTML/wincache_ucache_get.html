wincache_ucache_get

Example

 mixed wincache_ucache_get ( mixed $key [, bool &$success ] )

Description

Gets a variable stored in the user cache.

Return Values

If key is a string, the function returns the value of the variable stored with that key. The success is set to TRUE on success and to FALSE on failure. The key is an array, the parameter success is always set to TRUE. The returned array (name => value pairs) will contain only those name => value pairs for which the get operation in user cache was successful. If none of the keys in the key array finds a match in the user cache an empty array will be returned.
