xmlrpc_set_type

Example

 bool xmlrpc_set_type ( string &$value , string $type )

Description

Sets xmlrpc type, base64 or datetime, for a PHP string value. Warning: This function is EXPERIMENTAL. The behaviour of this function, its name, and surrounding documentation may change without notice in a future release of PHP. This function should be used at your own risk.

Return Values

Returns TRUE on success or FALSE on failure. If successful, value is converted to an object.
