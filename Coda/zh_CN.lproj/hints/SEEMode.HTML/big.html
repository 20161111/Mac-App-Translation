<big>

displays text in larger type

Deprecated

Example

What is <big>that?</big>

Description

<big> is unsupported in HTML5. Use CSS for styling instead.

<big> was used to create spans of text which rendered in a font size larger than the document's base size.

  • H4-FRAMES
  • H4-STRICT
