HttpRequest::getHistory

Example

 public HttpMessage HttpRequest::getHistory ( void )

Description

Get all sent requests and received responses as an HttpMessage object. If you want to record history, set the instance variable HttpRequest::recordHistory to TRUE. The returned object references the last received response, use HttpMessage::getParentMessage() to access the data of previously sent requests and received responses.

Return Values

Returns an HttpMessage object representing the complete request/response history.
