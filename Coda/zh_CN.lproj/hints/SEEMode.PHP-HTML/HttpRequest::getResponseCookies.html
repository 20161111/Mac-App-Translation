HttpRequest::getResponseCookies

Example

 public array HttpRequest::getResponseCookies ([ int $flags = 0 [, array $allowed_extras ]] )

Description

Get response cookie(s) after the request has been sent. If redirects were allowed and several responses were received, the data references the last received response.

Return Values

Returns an array of stdClass objects like http_parse_cookie() would return.
