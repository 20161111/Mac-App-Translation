HttpResponse::setThrottleDelay

Example

 static bool HttpResponse::setThrottleDelay ( float $seconds )

Description

Sets the throttle delay. Note: Provides a basic throttling mechanism, which will yield the current process or thread until the entity has been completely sent. Note: This may not work as expected with the following SAPI(s): FastCGI.

Return Values

Returns TRUE on success or FALSE on failure.
