com_load_typelib

Example

 bool com_load_typelib ( string $typelib_name [, bool $case_insensitive = true ] )

Description

Loads a type-library and registers its constants in the engine, as though they were defined using define(). Note that it is much more efficient to use the configuration setting to pre-load and register the constants, although not so flexible. If you have turned on , then PHP will attempt to automatically register the constants associated with a COM object when you instantiate it. This depends on the interfaces provided by the COM object itself, and may not always be possible.

Return Values

Returns TRUE on success or FALSE on failure.
