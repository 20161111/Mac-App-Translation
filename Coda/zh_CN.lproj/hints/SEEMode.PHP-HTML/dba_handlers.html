dba_handlers

Example

 array dba_handlers ([ bool $full_info = false ] )

Description

List all the handlers supported by this extension.

Return Values

Returns an array of database handlers. If full_info is set to TRUE, the array will be associative with the handlers names as keys, and their version information as value. Otherwise, the result will be an indexed array of handlers names. Note: When the internal cdb library is used you will see cdb and cdb_make.
