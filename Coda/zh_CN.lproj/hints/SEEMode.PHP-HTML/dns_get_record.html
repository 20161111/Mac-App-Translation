dns_get_record

Example

 array dns_get_record ( string $hostname [, int $type = DNS_ANY [, array &$authns [, array &$addtl ]]] )

Description

Fetch DNS Resource Records associated with the given hostname.

Return Values

This function returns an array of associative arrays, or FALSE on failure. Each associative array contains at minimum the following keys: Basic DNS attributes Attribute Meaning
