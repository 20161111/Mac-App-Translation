fgetc

Example

 string fgetc ( resource $handle )

Description

Gets a character from the given file pointer.

Return Values

Returns a string containing a single character read from the file pointed to by handle. Returns FALSE on EOF. Warning: This function may return Boolean FALSE, but may also return a non-Boolean value which evaluates to FALSE, such as 0 or "". Please read the section on Booleans for more information. Use the === operator for testing the return value of this function.
