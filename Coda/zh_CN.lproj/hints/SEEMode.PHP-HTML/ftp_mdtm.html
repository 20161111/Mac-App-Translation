ftp_mdtm

Example

 int ftp_mdtm ( resource $ftp_stream , string $remote_file )

Description

ftp_mdtm() gets the last modified time for a remote file. Note: Not all servers support this feature! Note: ftp_mdtm() does not work with directories.

Return Values

Returns the last modified time as a Unix timestamp on success, or -1 on error.
