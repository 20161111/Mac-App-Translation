geoip_id_by_name

Example

 int geoip_id_by_name ( string $hostname )

Description

The geoip_id_by_name() function will return the Internet connection type corresponding to a hostname or an IP address. The return value is numeric and can be compared to the following constants: GEOIP_UNKNOWN_SPEED GEOIP_DIALUP_SPEED GEOIP_CABLEDSL_SPEED GEOIP_CORPORATE_SPEED

Return Values

Returns the connection type.
