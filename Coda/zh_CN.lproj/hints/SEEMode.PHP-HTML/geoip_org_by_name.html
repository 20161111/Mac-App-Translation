geoip_org_by_name

Example

 string geoip_org_by_name ( string $hostname )

Description

The geoip_org_by_name() function will return the name of the organization that an IP is assigned to. This function is currently only available to users who have bought a commercial GeoIP Organization, ISP or AS Edition. A warning will be issued if the proper database cannot be located.

Return Values

Returns the organization name on success, or FALSE if the address cannot be found in the database.
