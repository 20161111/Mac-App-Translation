geoip_region_by_name

Example

 array geoip_region_by_name ( string $hostname )

Description

The geoip_region_by_name() function will return the country and region corresponding to a hostname or an IP address. This function is currently only available to users who have bought a commercial GeoIP Region Edition. A warning will be issued if the proper database cannot be located. The names of the different keys of the returning associative array are as follows: "country_code" -- Two letter country code (see geoip_country_code_by_name()) "region" -- The region code (ex: CA for California)

Return Values

Returns the associative array on success, or FALSE if the address cannot be found in the database.
