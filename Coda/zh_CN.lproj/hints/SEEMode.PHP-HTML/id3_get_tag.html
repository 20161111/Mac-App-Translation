id3_get_tag

Example

 array id3_get_tag ( string $filename [, int $version = ID3_BEST ] )

Description

id3_get_tag() is used to get all information stored in the id3 tag of the specified file.

Return Values

Returns an associative array with various keys like: title, artist, .. The key genre will contain an integer between 0 and 147. You may use id3_get_genre_name() to convert it to a human readable string.
