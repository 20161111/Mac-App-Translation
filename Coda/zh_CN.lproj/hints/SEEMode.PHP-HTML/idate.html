idate

Example

 int idate ( string $format [, int $timestamp = time() ] )

Description

Returns a number formatted according to the given format string using the given integer timestamp or the current local time if no timestamp is given. In other words, timestamp is optional and defaults to the value of time(). Unlike the function date(), idate() accepts just one char in the format parameter.

Return Values

Returns an integer. As idate() always returns an integer and as they can't start with a "0", idate() may return fewer digits than you would expect. See the example below.
