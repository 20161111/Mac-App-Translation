imap_body

Example

 string imap_body ( resource $imap_stream , int $msg_number [, int $options = 0 ] )

Description

imap_body() returns the body of the message, numbered msg_number in the current mailbox. imap_body() will only return a verbatim copy of the message body. To extract single parts of a multipart MIME-encoded message you have to use imap_fetchstructure() to analyze its structure and imap_fetchbody() to extract a copy of a single body component.

Return Values

Returns the body of the specified message, as a string.
