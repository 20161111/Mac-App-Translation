imap_fetchstructure

Example

 object imap_fetchstructure ( resource $imap_stream , int $msg_number [, int $options = 0 ] )

Description

Fetches all the structured information for a given message.

Return Values

Returns an object includes the envelope, internal date, size, flags and body structure along with a similar object for each mime attachment. The structure of the returned objects is as follows: Returned Objects for imap_fetchstructure() type Primary body type
