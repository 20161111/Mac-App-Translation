ingres_fetch_row

Example

 array ingres_fetch_row ( resource $result )

Description

ingres_fetch_row() returns an array that corresponds to the fetched row, or FALSE if there are no more rows. Each result column is stored in an array offset, starting at offset 1. Subsequent calls to ingres_fetch_row() return the next row in the result set, or FALSE if there are no more rows. By default, arrays created by ingres_fetch_row() start from position 1 and not 0 as with other DBMS extensions. The starting position can be adjusted to 0 using the configuration parameter ingres.array_index_start. Note: Related Configurations See also the ingres.array_index_start, ingres.fetch_buffer_size and ingres.utf8 directives in Runtime Configuration.

Return Values

Returns an array that corresponds to the fetched row, or FALSE if there are no more rows
