is_scalar

Example

 bool is_scalar ( mixed $var )

Description

Finds whether the given variable is a scalar. Scalar variables are those containing an integer, float, string or boolean. Types array, object and resource are not scalar. Note: is_scalar() does not consider resource type values to be scalar as resources are abstract datatypes which are currently based on integers. This implementation detail should not be relied upon, as it may change.

Return Values

Returns TRUE if var is a scalar FALSE otherwise.
