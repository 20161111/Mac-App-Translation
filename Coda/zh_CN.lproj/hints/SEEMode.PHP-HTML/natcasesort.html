natcasesort

Example

 bool natcasesort ( array &$array )

Description

natcasesort() is a case insensitive version of natsort(). This function implements a sort algorithm that orders alphanumeric strings in the way a human being would while maintaining key/value associations. This is described as a "natural ordering".

Return Values

Returns TRUE on success or FALSE on failure.
