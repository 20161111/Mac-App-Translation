odbc_columns

Example

 resource odbc_columns ( resource $connection_id [, string $qualifier [, string $schema [, string $table_name [, string $column_name ]]]] )

Description

Lists all columns in the requested range.

Return Values

Returns an ODBC result identifier or FALSE on failure. The result set has the following columns: TABLE_QUALIFIER TABLE_SCHEM TABLE_NAME COLUMN_NAME DATA_TYPE TYPE_NAME PRECISION LENGTH SCALE RADIX NULLABLE REMARKS The result set is ordered by TABLE_QUALIFIER, TABLE_SCHEM and TABLE_NAME.
