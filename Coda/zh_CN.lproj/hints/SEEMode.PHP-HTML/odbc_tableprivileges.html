odbc_tableprivileges

Example

 resource odbc_tableprivileges ( resource $connection_id , string $qualifier , string $owner , string $name )

Description

Lists tables in the requested range and the privileges associated with each table.

Return Values

An ODBC result identifier or FALSE on failure. The result set has the following columns: TABLE_QUALIFIER TABLE_OWNER TABLE_NAME GRANTOR GRANTEE PRIVILEGE IS_GRANTABLE The result set is ordered by TABLE_QUALIFIER, TABLE_OWNER and TABLE_NAME.
