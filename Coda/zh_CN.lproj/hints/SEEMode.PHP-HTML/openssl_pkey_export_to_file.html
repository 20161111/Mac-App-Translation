openssl_pkey_export_to_file

Example

 bool openssl_pkey_export_to_file ( mixed $key , string $outfilename [, string $passphrase [, array $configargs ]] )

Description

openssl_pkey_export_to_file() saves an ascii-armoured (PEM encoded) rendition of key into the file named by outfilename. Note: You need to have a valid openssl.cnf installed for this function to operate correctly. See the notes under the installation section for more information.

Return Values

Returns TRUE on success or FALSE on failure.
