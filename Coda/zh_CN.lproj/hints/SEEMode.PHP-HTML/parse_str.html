parse_str

Example

 void parse_str ( string $str [, array &$arr ] )

Description

Parses str as if it were the query string passed via a URL and sets variables in the current scope. Note: To get the current QUERY_STRING, you may use the variable $_SERVER['QUERY_STRING']. Also, you may want to read the section on variables from external sources. Note: The magic_quotes_gpc setting affects the output of this function, as parse_str() uses the same mechanism that PHP uses to populate the $_GET, $_POST, etc. variables.

Return Values

No value is returned.
