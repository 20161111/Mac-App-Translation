pg_fetch_all

Example

 array pg_fetch_all ( resource $result )

Description

pg_fetch_all() returns an array that contains all rows (records) in the result resource. Note: This function sets NULL fields to the PHP NULL value.

Return Values

An array with all rows in the result. Each row is an array of field values indexed by field name. FALSE is returned if there are no rows in the result, or on any other error.
