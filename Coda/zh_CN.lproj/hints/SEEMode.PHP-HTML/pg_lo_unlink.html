pg_lo_unlink

Example

 bool pg_lo_unlink ( resource $connection , int $oid )

Description

pg_lo_unlink() deletes a large object with the oid. Returns TRUE on success or FALSE on failure. To use the large object interface, it is necessary to enclose it within a transaction block. Note: This function used to be called pg_lounlink().

Return Values

Returns TRUE on success or FALSE on failure.
