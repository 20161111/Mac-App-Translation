readdir

Example

 string readdir ([ resource $dir_handle ] )

Description

Returns the name of the next entry in the directory. The entries are returned in the order in which they are stored by the filesystem.

Return Values

Returns the entry name on success or FALSE on failure. Warning: This function may return Boolean FALSE, but may also return a non-Boolean value which evaluates to FALSE, such as 0 or "". Please read the section on Booleans for more information. Use the === operator for testing the return value of this function.
