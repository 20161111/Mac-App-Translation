Example

 string sqlite_fetch_single ( resource $result [, bool $decode_binary = true ] )

Description

Object oriented style (method): string SQLiteResult::fetchSingle ([ bool $decode_binary = true ] )
string SQLiteUnbuffered::fetchSingle ([ bool $decode_binary = true ] )
sqlite_fetch_single() is identical to sqlite_fetch_array() except that it returns the value of the first column of the rowset. This is the most optimal way to retrieve data when you are only interested in the values from a single column of data.

Return Values

Returns the first column value, as a string.
