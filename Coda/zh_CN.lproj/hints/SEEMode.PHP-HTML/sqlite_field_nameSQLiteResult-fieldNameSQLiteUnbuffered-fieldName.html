sqlite_field_nameSQLiteResult-fieldNameSQLiteUnbuffered-fieldName

Example

 string sqlite_field_name ( resource $result , int $field_index )

Description

Object oriented style (method): string SQLiteResult::fieldName ( int $field_index )
string SQLiteUnbuffered::fieldName ( int $field_index )
Given the ordinal column number, field_index, sqlite_field_name() returns the name of that field in the result set result.

Return Values

Returns the name of a field in an SQLite result set, given the ordinal column number; FALSE on error. The column names returned by SQLITE_ASSOC and SQLITE_BOTH will be case-folded according to the value of the sqlite.assoc_case configuration option.
