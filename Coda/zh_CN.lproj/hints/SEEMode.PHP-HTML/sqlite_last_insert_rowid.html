sqlite_last_insert_rowid

Example

 int sqlite_last_insert_rowid ( resource $dbhandle )

Description

Returns the rowid of the row that was most recently inserted into the database dbhandle, if it was created as an auto-increment field. Tip You can create auto-increment fields in SQLite by declaring them as INTEGER PRIMARY KEY in your table schema.

Return Values

Returns the row id, as an integer.
