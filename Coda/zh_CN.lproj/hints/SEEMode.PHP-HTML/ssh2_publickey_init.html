ssh2_publickey_init

Example

 resource ssh2_publickey_init ( resource $session )

Description

Request the Publickey subsystem from an already connected SSH2 server. The publickey subsystem allows an already connected and authenticated client to manage the list of authorized public keys stored on the target server in an implementation agnostic manner. If the remote server does not support the publickey subsystem, the ssh2_publickey_init() function will return FALSE.

Return Values

Returns an SSH2 Publickey Subsystem resource for use with all other ssh2_publickey_*() methods or FALSE on failure.
