str_word_count

Example

 mixed str_word_count ( string $string [, int $format = 0 [, string $charlist ]] )

Description

Counts the number of words inside string. If the optional format is not specified, then the return value will be an integer representing the number of words found. In the event the format is specified, the return value will be an array, content of which is dependent on the format. The possible value for the format and the resultant outputs are listed below. For the purpose of this function, 'word' is defined as a locale dependent string containing alphabetic characters, which also may contain, but not start with "'" and "-" characters.

Return Values

Returns an array or an integer, depending on the format chosen.
