swf_mulcolor

Example

 void swf_mulcolor ( float $r , float $g , float $b , float $a )

Description

Sets the global multiply color to the given one. This color is then implicitly used by the swf_placeobject(), swf_modifyobject() and swf_addbuttonrecord() functions. The color of the object will be multiplied by the given color values when the object is written to the screen.

Return Values

No value is returned.
