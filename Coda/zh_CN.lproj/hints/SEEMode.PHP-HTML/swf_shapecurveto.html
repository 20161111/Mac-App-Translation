swf_shapecurveto

Example

 void swf_shapecurveto ( float $x1 , float $y1 , float $x2 , float $y2 )

Description

Draws a quadratic bezier curve from the current location, though the two given points. The current position is then set to the point defined by the x2 and y2 parameters.

Return Values

No value is returned.
