sybase_close

Example

 bool sybase_close ([ resource $link_identifier ] )

Description

sybase_close() closes the link to a Sybase database that's associated with the specified link link_identifier. Note that this isn't usually necessary, as non-persistent open links are automatically closed at the end of the script's execution. sybase_close() will not close persistent links generated by sybase_pconnect().

Return Values

Returns TRUE on success or FALSE on failure.
