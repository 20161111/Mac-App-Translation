    Drag and drop

The easiest way to add data to DEVONthink Pro Office, other than creating the documents directly inside the application, is to drag files or selected data, e.g. text, images, etc., into a DEVONthink Pro Office window or onto its Dock icon.

Dragging files into a document list imports the file; dragging text or images creates documents for them. Dragging files into a rich text document adds them as attachments to the document. Dragging data to the Dock icon creates new documents in the default destination.

Hold the ⌘Command key while dragging to effectively move the file or folder into the database. DEVONthink Pro Office imports the dragged item and puts the original into the Finder trash. This works identical to File > Import > Files & Folders with the option Move originals to trash enabled.
