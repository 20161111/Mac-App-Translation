How to generate new glyphs

There are several ways to generate new glyphs

Manually generate new glyphs

  1. Press ⌘+⇧+G

  2. In the up comming dialog, enter the glyph names for the glyphs to generate and press OK.

Generate glyphs to fill a language group.

  1. Select the Font View

  2. In the sidebar, select the language group (e.g. Latin > Wester European) you want to add glyphs to. You see the missing glyphs count in the badge.

  3. Right click the row.

  4. In the popup window, select all glyphs you want to generate and hit Generate.
