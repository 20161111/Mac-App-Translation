Accented Letters and Language Coverage

This shall explain how to add all the additional letters to support more languages.

Glyphs contains a data base that contains informations about letters and glyphs. And it contains a list about what letters are necessary in each language.

Preparing the font

You need to draw all base glyphs. For Latin languages that includes the letters A-Z and a-z and several accents like acute, grave and dieresis (check CATEGORIES > Marks > Spacing)

Add anchors to this glyphs. See: Composites, Components and Anchors

Adding the glyphs

In the Font View, in the sidebar, choose the category or language you what to support. All entries that have a badge with a number next to it, you can right click it and choose the letters you want to add from the list and hit the generate button.

Glyphs will add all letters and try to use composites as much as possible. All empty letters will show a gray preview of the shape. This is only an orientation and no design recommendation.

Fill in all the missing glyphs and check the accent positions.
