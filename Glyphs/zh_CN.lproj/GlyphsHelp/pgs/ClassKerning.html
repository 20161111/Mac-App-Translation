Class Kerning in Glyphs

In Glyphs, kerning works like in Opentype fonts or in Metrics Machine. Kerning is applied to glyphs or classes. There is no such thing like key glyphs.

Each glyph can be member of a left kerning class (it deals with the kerning on the left side of the glyph) and a right kerning class. To add a glyph to the kerning class, you add the name of the class to the glyphs left and right kerning class fields. All glyphs with the same class names are placed in the same group.

The name of the class can be any string, but it is recommended to use the a descriptive name like the name of the base glyph.

Each kerning pair can be kerned according to the class or can be an exception and have its own value. To change it, click the look icon in the glyph inspector.
