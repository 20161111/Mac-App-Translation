Editing Paths

Add Paths

  1. Use the Draw tool the add nodes. Simple click adds a line, click and drag adds a curve.

Select and move

  1. You can select and move nodes and paths like you are used to it.

  2. Double click a path to select it.

Editing options

  1. If you move a off curve node with the mouse or the keyboard and hold down ⌥ (option), the angle is preserved.

  2. If you move a smooth curve node with the mouse or the keyboard and hold down ⌥ (option), movement is constrained to the axis of the two offcurve point.

  3. To break a path, click a node with the Draw Tool.

  4. To join two open paths, move the end nodes on top of each other.

Deleting points

  1. If only one smooth curve point is selected and you delete it (with ⌫) the appearance of the path is retained as much as possible.

  2. if you delete a offcurve point, the segment is converted into a strait line.

Convert Line segment to Curve

  1. Click the line segment while holding down ⌥.

Other Things

  1. Select one corner point, and select "Open Corner" from the context menu.

  2. Select two point, and select "Reconnect Nodes" from the context menu.
