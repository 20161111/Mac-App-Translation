Export OpenType Fonts

Glyphs can export OpenType Fonts. It uses a makeOTF from Adobe FDK to compile them.

If you have a Multi Layer Font, the instances are exported.

Options on export

  1. Use Production glyph names

    This converts all readable names production names (e.g Acyrillic to afii10017, Tcommaaccent to uni021A)

  2. Remove Overlap

    This decomposes all glyphs and removes overlap. The first run takes a bit more time.

  3. Autohint unhinted glyphs

    This uses Adobe autohint to hint all glyphs without hints. It needs stem values to be defined in the the Master Setting.

  4. Install Font

    To copy the exported font to the specified location. For example, set this to the Adobe Fonts folder (/Library/Application Support/Adobe/Font) for easy testing of fonts in InDesignÂ®.
