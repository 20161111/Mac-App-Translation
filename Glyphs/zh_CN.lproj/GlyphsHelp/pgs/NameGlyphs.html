Defining glyph names

There are several ways to set the glyph name.

Glyph names are automatically converted to a readable name (e.g. if you input "afii10017", it will be converted to "Acyrillic").

If you don’t know the exact glyph name but the unicode value, you can use "uni"+unicode index. (e.g. uni0410 for the cyrillic uppercase "A").

In the Font View

  1. Select the glyph in the Font view

  2. Hit ↵ (enter) or click on the glyph name.

    While typing, a popup will suggest a valid name.

In the Edit View

  1. Select the glyph

  2. Remove any selection if you are in selection Mode

  3. In the inspector pane, click on the glyph name.

    While typing, a popup will suggest a valid name.

In the Font View

  1. Select one or more glyphs in the Font View

  2. From the menu choose "Edit" > "Find" > "Search And Replace".

    In the dialog, put in the search string and replace string.
