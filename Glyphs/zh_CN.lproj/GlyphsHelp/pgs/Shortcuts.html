Keyboard shortcuts

This is a list of keyboard shortcuts for Glyphs.
If you don’t find something here, try to find the command and the shortcut in the menu.

General

 

Select a Master

⌘+⌥+1..x (the number of the Master)

Font View

 

Open single glyph in new edit tab

⌘+↓, or double click

Open multiple glyphs in new edit tab

⌘+↓, or ⌘+double click

Edit the glyph name of a single selected glyph

↵, or single click on glyph's name

Add new glyph

⌘+⌥+⇧+N

Select the search field

⌘+F

Duplicate selected glyphs

⌘+D

Copy selected glyphs

⌘+C

Paste selected glyphs

⌘+V

Delete selected glyphs

⌘+Del

Open Glyph Properties

Right click on glyph

Edit View

 

Selection Tool

V or esc if you are in the text mode, or hold ⌘ if any other tool is selected

Select all nodes

⌘+A

Select Path

Double click on any segment

Last Filter Used

⌘+R

Select all nodes and anchors

⌘+A twice

Toggle node’s mode (smooth or corner)

Double click the node or hit enter with on node selected.

Move node, but keep handles in place

⌥ + click and drag

Draw Tool

P

Primitives

F, click and hold icon to change primitive shape

Rotate Tool

R

Scale Tool

S

Text Tool

T

Hand Tool

H or hold space bar

Zoom Tool

Z

Zoom In

⌘++, ⌘+Space Bar+Click, and ⌥+Scroll Up

Zoom Out

⌘+-, ⌘+⌥+Space Bar+Click, and ⌥+Scroll Down

Measurement Tool

L

Preview current glyph

Hold space bar

Switch to previous glyph

Home or Fn+← on MacBook keyboard

Switch to next glyph

End or Fn+→ on MacBook keyboard

Choose a glyph not accessible from the keyboard

⌘+F

Activate Background Layer

⌘+B

Set the Selection as Background

⌘+K

Zoom in/out

⌘+Space/⌘+⌥+Space or ⌘++/-

Hand Tool

Space

Correct Paths Direction

⌘+⇧+R

Open a path

Click on a node with the Draw Tool selected

Close a path

Drag an end node on top of another end node

Add handles to node

⌥+click on segment next to node

Select Next Node/Handle

Tab

Select Previous Node/Handle

⇧+Tab

Center align anchor between two nodes

Select two nodes and an anchor, press ⌘+⇧+A

Align nodes/handles

⌘+⇧+A

Preview accent position

Click on any anchor

Move object/node in any direction by 1 unit

↑, ↓, ←, or →

Move object/node in any direction by 10 units

⇧+↑, ↓, ←, or →

Move object/node in any direction by 100 units

⌘+↑, ↓, ←, or →

Add Component

⌘+⇧+C

Make Component Glyph

⌘+⇧+⌥+C

Update Metrics

⌃+⌘+M

Edit Background

⌘+B

Copy Selection to Background Layer

Show/Hide Background Layer

⌘+⇧+B

Reverse Contours

⌘+⇧+⌥+R

Correct Path Direction

⌘+⇧+R

Tidy Up Paths

⌘+⇧+⌥+T

Type Tool

 

To insert text just type.

 

Create new Text tab

⌘+T

Insert glyphs not accessible from keyboard

⌘+F

Insert sample text

⌘+⌥+F

Adjusting the spacing of the selected glyphs.

⌘+← / ⌘+→ for right sidebearing
⌃+← / ⌃+→ for left sidebearing
use ⇧ (⇧) to move 10 units

Adjusting the kerning of the selected glyphs.

⌘+⌥+← / ⌘+⌥+→ for the kerning with the letter to the right
⌃+⌥+← / ⌃+⌥+→ for the kerning with the letter to the left
use ⇧ (⇧) to move 10 units

Select Tool

 

To keep the the angle of an offcurve point, hold down the ⌥ (option) key while dragging or using the arrow keys.

⌥ + mouse or arrow key

To only move the smooth curve point, hold down the ⌥ (option) key while dragging or using the arrow keys.

⌥ + mouse or arrow key

Select All Nodes + Other Elements (anchors...)

⌘+A (twice)

Select a path

Double Click path

Change smoothing setting of Node

Double click a node or select a node and hit ↵ (return)
