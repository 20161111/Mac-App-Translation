Fonts

You can use the Fonts pane of the Cleaning toolbar item or the Automation pane to remove the fonts cache files.

You can use these files to accelerate font loading. Deleting them will temporarily slow the fonts from appearing, but it will also correct some issues. This will not delete any personal font collection.

■

From the Fonts pane of the Cleaning toolbar item, you can independently remove the fonts cache for the System, Users, and Apple applications; Adobe; Microsoft; Quark; and some open source applications.

■

From the Automation pane, you can delete all fonts caches.

After cleaning the fonts cache of some applications, the initial startup can be particularly lengthy.
