Misc.

You can delete various files using the Misc. pane of the Cleaning toolbar item.

■

Automatically saved versions of the documents: You can use this option to delete all the saved versions of your documents.

More information about Auto Save . . .

■

Saved application states: You can use this option to delete the saved state of all applications.

More information about saved application states . . .

■

Recent items: You can use this option to remove the Recent Folders in the Finder's Go menu, the Recent Servers in Terminal, the Recent Places in the Open and Save dialogs, the Recent Items in the Open Recent command in the File menu of applications, and the appearance of Recent Items (files, folders, searches, servers, and so on) of third–party applications.

■

Recent conversions in Calculator: You can use this option to delete the conversions recently used in Calculator.

■

Items in the Mail Downloads folder: This folder, situated in the Library folder of your home folder, contains a copy of files opened in the Mail application. To remove these copies, select this option.

■

Items in the Saved Searches folder: In the Finder's windows, you can save your searches as a Smart Folder. You can use this option to remove these saved searches located in the default folder. The items in the Sidebar are not deleted.

■

Obsolete items: Under some conditions, a copy of the application state when an application quits unexpectedly is stored in core.xxx files located in the invisible /cores/ folder. You can use this option to remove all core.xxx files. This option also deletes all .plist.lockfile files that may be present in different Preferences folders.

■

Previous iTunes Libraries: When you open a new version of iTunes, the existing library is saved in a Previous iTunes Libraries folder and then it is upgraded. If the library is upgraded correctly, the old library can be deleted with this option. This may make several MB of disk space available.

■

My Computers list in Screen Sharing: You can use this option to delete the list located in the My Computers section of the Screen Sharing application.

■

QuickTime Content Guide: By default, opening QuickTime Player 7 displays the content listing in entertainment on the Internet (also available from the Show Content Guide command of the Window menu). You can use this option to delete the cache files of this content.

■

Contents of the Xcode Derived Data folder: You can use this option to delete the contents of the DerivedData folder used by Xcode.
