User

You can use the User pane of the Cleaning toolbar item or the Automation pane to remove any cache files (applications and preferences caches, Dock cache, and so on) and temporary items. The deletion of these files recovers some disk space and prevents a damaged file from impacting computer functions.

You should run this command occasionally. A slight temporary decrease in speed will be observed (when applications are open, displaying System Preferences, searching for updates, and so on).

The cleaning of the user cache from the Automation pane doesn't remove the Audio Units cache.
