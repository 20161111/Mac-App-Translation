Web browser cache and history

You can use the options in the Internet pane of the Cleaning toolbar item to remove the following items:

■

Browser cache: These are files created when loading Internet pages in the web browser.

■

Download cache: These are cache files created when files are downloaded from the web.

■

Browser history: This is the list of pages that web browsers keep to make pages load more quickly.

■

Recent searches: These are recent searches that the web browsers store.

■

Webpage previews: These are preview images from the webpages you’ve visited most recently in the Top Sites page of Safari.

These browsers are supported: Aviator, Browser, Chromium, Cruz, Desktop Web Browser, Fake, Firefox, Google Chrome, Helium, iCab, iTubeX, Maxthon, OmniWeb, Opera, PolyBrowser, Roccat, Safari, SeaMonkey, Sleipnir, SRWare Iron, Vivaldi, WebKit, and xStand.
