Problem with selecting items on the desktop

If, on the desktop, you cannot select multiple files and folders by dragging a box around them using the mouse, you probably turned on the "Use the Simple Finder instead of the full Finder" function. To correct this issue, you must turn off this feature:

Open OnyX, and enter your password.

Click the Parameters toolbar button, and select the Finder tab.

Deselect the "Use the Simple Finder instead of the full Finder" box.

However, if you want to use the Simple Finder instead of the full Finder, then you must change your habits. You can select multiple desktop items by clicking each while holding down the Shift or Command key.

Do not forget: If a function is hidden, there must be a good reason.
