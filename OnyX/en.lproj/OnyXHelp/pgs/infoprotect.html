Protection

The Protection pane in the Info toolbar item gives some info about XProtect, the integrated protection system of OS X. XProtect checks each new file downloaded from your web browser, received by email, or received by instant messaging to protect the system from any malware (or malicious software).

- The list on the left displays the list of recognized XProtect malware. By default, OS X regularly searches for new malware definitions (every 24 hours), and the list is updated if newer definitions are available.

Important: To perform a daily check for new malware definitions, the "Automatically check for updates" and "Install system data files and security updates" options in the App Store pane of System Preferences must be selected.

- The list on the right displays a list of all downloaded items tested by XProtect.

Important: If you select the "Remove the alert message on first opening applications downloaded from the Internet" option from the Parameters > Safari pane, XProtect will not check the downloaded files anymore.
