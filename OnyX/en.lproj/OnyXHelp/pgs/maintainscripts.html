Scripts

OS X includes a task manager (launchd) that automates system cleaning by executing the "periodic" utility. This utility runs three BSD standard scripts that perform many maintenance operations.

To run the Scripts:

Select the Maintenance toolbar button, and click the Scripts tab.

Select the scripts you want to execute (daily, weekly, and monthly).

Click the Execute button.

You can use the Automation pane to run all maintenance scripts at once.

The lower section of the Scripts pane of the Maintenance toolbar item manages the logs of maintenance scripts (daily.out, weekly.out, and monthly.out files located in the /private/var/log folder). You can show the content, the path, and the size of each log. You can also individually or simultaneously clean the logs.

The Clean and Clean All buttons delete the content of logs but don't delete the files. If the Clean button is dimmed, this means that no log is selected or the selected log doesn't exist. (The corresponding scripts have probably never been executed.)
