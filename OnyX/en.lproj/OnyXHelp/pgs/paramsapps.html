Applications

You can use the Applications pane of the Parameters toolbar item to set parameters for the hidden functions of the applications.

Disk Utility:

■

Verify disk images (.dmg): You can use this option to turn on or off the verification when you open disk images.

■

Verify the disks after burning: You can use this option to turn on or off verification after burning a disk.

Screen Sharing:

■

Show the Bonjour browser: You can use this option to show or hide the list of computers available to the Apple Bonjour protocol.

■

Show the full toolbar: You can use this option to show or hide the full toolbar while screen sharing.

QuickTime Player:

■

Autoplay movies: Select this option if you want to automatically play movies after opening them.

■

Show QuickTime Player's Movie Inspector when QuickTime Player 7 is opened: You can use this option to automatically display QuickTime Player 7's Inspector window when you open QuickTime Player 7 or when you open a file. This only works with QuickTime Player version 7.

App Store:

■

Show the App Store Debug menu: This option shows or hides a Debug menu in the App Store menu bar.

■

Automatically install new malware definitions: Select this option to automatically perform a daily search and install all available security updates including new malware definitions used by XProtect, the integrated protection system of OS X.

Mission Control:

■

Turn on Mission Control: This option turns on and turns off Mission Control and all features.

■

Turn on graphic effect: When you run or quit Mission Control, a zoom or minimize effect appears. You can use this option to turn on or turn off this graphic effect.

X11:

■

Show an alert message before quitting the application: You can use this option to show or hide the alert message that appears by default when quitting any X11 application.

■

At opening of X11, execute: You can use this menu to choose the application that will load automatically when you open X11.

Launchpad:

■

Reset: This button deletes the icons in Launchpad then reloads the applications and displays their icons.

Misc. applications:

■

Show the Reminders Debug menu: This option shows or hides a Debug menu in the Reminders menu bar.

■

Turn on the Terminal window focus: You can turn on this option to add the ability to choose which Terminal window is active by placing the mouse pointer above it.

■

Show Quartz Debug's hidden options: You can use this option to add supplementary options in the Tools and Window menus of Quartz Debug.

■

Turn on the Dashboard keyboard shortcut: If you turn on this option, clicking on the Dashboard keyboard shortcut (F12 by default) will display widgets.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by any other application. After execution, the parameters will be reset, and the display will be as it was when the system and the applications were first installed.
