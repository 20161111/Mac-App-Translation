iTunes

You can use the iTunes pane of the Parameters toolbar item to set parameters for the hidden functions of iTunes.

Importing:

■

Add Track Numbers: By default, while importing the songs in iTunes, the name of the created file starts with the track number. You can use this option to display or not display the track number in the name of created files while importing.

■

Automatically play the songs: You can use this option to automatically begin the reading of the imported songs. Generally, the reading only starts after importing the first song.

Misc. options:

■

Turn on the rating by half-star: You can assign a rating to a song, album, video, podcast, or other item to indicate how much you like it. You can turn on this option to specify the rating with a half-star. Click just beside a point or a star to assign an intermediate notation (1/2).

■

Turn on the High Contrast mode: This option replaces the background to the left side of the window as well as all the lists with a black background, which improves the contrast.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by any other application. After execution, the parameters will be reset, and the display will be as it was when the system and the applications were first installed.
