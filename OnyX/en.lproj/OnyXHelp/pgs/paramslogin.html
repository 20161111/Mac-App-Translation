Login

You can use the Login pane of the Parameters toolbar item to set parameters for the miscellaneous options of the login window.

Appearance:

You can change the background image of the login window.

■

Background: By default, the image background of the login window is the same as the Desktop picture, but it is blurred. You can use the Select button to select a replacement image. (The application directs you to the folder containing the desktop pictures of OS X.)

These formats are accepted: BMP, GIF, JPEG, JPEG 2000, PNG, PSD, TGA, and TIFF.

You can use the Restore option to put the original background image in the login window (the Desktop blurred picture). However, the login window and the preview in OnyX will display the default image installed by the system (El Capitan). A restart is required to display the correct image.

Options:

■

Hide the Sleep, Restart, and Shut Down commands in the login window: This option hides the Sleep, Restart, and Shut Down buttons located at the bottom of the login window.

■

Show message in the login window: You can use this option to display text in the login window. Select this option; then enter the text in the field located below, and click the Apply button. Warning: This function doesn't accept certain special characters.

■

Show in the list of users: You can use this option to show or hide the users (Administrator, Local, and Network) in the list of users in the login window.

Warning:
- If the kiosk setting is turned on, don't deselect the administrator accounts in the list of the users. The kiosk mode makes the Restart, Shut Down, and Log Out commands unavailable in the Apple menu (). (See the Finder tab.)
- Turning off the local users in the list of users will display only the Guest account in the login window. To redisplay the local users, restart in single-user mode (Command-S at startup), and enter the three following commands (QWERTY keyboard):

/sbin/mount -uw /
rm /Library/Preferences/com.apple.loginwindow.plist
reboot

Startup mode:

You can use this menu to set the startup mode of the computer.

■

Normal: Normal mode displays a dark gray Apple logo (the apple) on a light gray background or a white Apple logo on a black background, shows a progress indicator (a moving bar) while various initializations are carried out, and then displays the desktop in the Finder.

■

Safe mode: Safe mode is the state OS X is in after a Safe Boot. Starting up into Safe mode forces a directory check of the startup disk, loads only required kernel extensions, runs only Apple–installed startup items, turns off all fonts other than those installed by the system, moves all font caches to the Trash, and turns off any login items. Safe mode can be useful for troubleshooting. However, certain OS X features do not work when you are started up in Safe mode. For example, you can't use DVD Player, capture video in iMovie, use an AirPort card, use audio input or output devices, or use an internal or external USB modem.

■

Verbose: Verbose mode displays a black screen with white text that shows details of the startup process line by line then displays the desktop in the Finder.

■

Single-user: Single-user mode displays a black screen with white text that shows details of the startup process line by line then displays a command prompt; to switch back to the OS X interface, enter the command nvram boot-args = "" . From the single-user mode, you can load the Finder by typing the command exit and thus switch back to the OS X interface.

WARNING: Your Mac can't start up in single-user or verbose mode if the computer owner or administrator has set a firmware password.
If FileVault is enabled, you need to unlock the startup disk as part of this process. White text appears briefly before the FileVault login screen is shown. After selecting a user and entering the user’s password, the single-user mode or verbose mode startup process continues.

Startup sound:

■

Startup sound: You can use this option to turn on or turn off the sound played at startup of the computer.

Warning: The startup sound provides information about the computer's hardware status.

Note: Some applications and system extensions (Soundflower, for example) can automatically turn on the startup sound again.

Power button:

■

Show the Shutdown dialog when the power button is pressed: By default, when the power button is pressed about one second, the computer falls into Sleep mode. You can use this option to instead display a Shutdown dialog from which you can restart, sleep, or shut down your Mac.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by any other application. After execution, all parameters will be reset and display as they did when the system and the applications were first installed.
