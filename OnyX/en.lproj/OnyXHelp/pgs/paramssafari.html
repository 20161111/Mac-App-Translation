Safari

You can use the Safari pane of the Parameters toolbar item to set the parameters of some hidden Safari functions.

Options:

■

Remove the alert message on first opening applications downloaded from the Internet: At first opening of an application downloaded from the Internet, an alert message appears. This message gives miscellaneous information on the source of the downloaded file, and you can open the file.

You can use this option to show or hide this alert message.
Important: Selecting this option presents a potential risk of security. Indeed, XProtect, the anti-malware system used by OS X, cannot check the downloaded files anymore.

■

Use the Delete key to go to the previous page in history: By default, to go to the previous web page, you must press the Command-Left Bracket keyboard shortcut. You can use this option to have the Delete key perform the same function.

Menus:

■

Turn on the Debug menu: This option shows or hides a Debug menu in the Safari menu bar.

■

Turn on the Develop menu: Safari has hidden features that can be turned on or off with this option. You can use this menu to open the current webpage in another browser, change the User Agent, display the Web Inspector, turn on or turn off some functions, and so on. This option also adds an Inspect Element item in the shortcut menu of Safari. By default, the Web Inspector appears in an individual window, but it can be merged with the displayed page by clicking the icon intended for this purpose located at the Inspector's bottom left.

Logs:

Enter the values that you want to use; then click the Apply button.

■

Number of items in the History menu: You can use this field to set the number of items appearing in the History menu in Safari.

■

Number of days: You can use this field to set the number of days preserved in the History of Safari.

Click the Restore Defaults button to cancel all modifications caused by OnyX or by another application. After execution, the parameters will display as they did when the system and the applications were first installed.
