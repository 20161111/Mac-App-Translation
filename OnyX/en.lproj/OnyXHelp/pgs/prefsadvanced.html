Preferences — Advanced

You can use the Advanced pane of OnyX preferences to set the verification options when OnyX opens.

■

By default, OnyX automatically checks the used disk space. If the used disk space is greater than or equal to 80 percent, OnyX displays an alert message advising to save space by deleting files. However, you can turn off this verification.

■

You can use this option to check the structure of the startup disk when OnyX opens. There are three available options (Ask, Automatically check, or Don't check).
