Preferences — Behavior

You can use the Behavior pane of OnyX preferences to modify the way OnyX functions.

■

You can choose to run the commands on the current account (by default) or on all accounts, if possible.

■

The "Before quitting applications" option will cause OnyX to show an alert message before it forces an application to quit.

■

The "After executing commands" option displays an information message after running each task. No message will be displayed if the "Automatically restart" or "Shut down" option is selected.

■

By default, when you should restart after running certain tasks, OnyX displays a message that asks what you want to do. You can choose to do nothing, restart, or shut down the Mac.

■

OnyX, like any open application, is automatically relaunched after a restart. If you don't want the application to relaunch, deselect the "Relaunch OnyX when computer restarts" option.

Click the Restore Defaults button to cancel your modifications and restore OnyX to its default setting.
