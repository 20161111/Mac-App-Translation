Preferences — Update

You can use the Update pane of OnyX preferences to set the OnyX update parameters and display the date and time OnyX last checked for an update.

■

To search, when OnyX opens, whether there is a more recent version of OnyX, select the "Check for updates when OnyX is opened" option. If a later version is available, it will be placed on your desktop (default option).

■

You can modify the save path: Click the Select button, and select the folder in the next dialog.

■

Click the Check button to determine if a more recent version is available.

Click the Restore Defaults button to cancel your modifications and restore OnyX to its default setting.
