System Integrity Protection

System Integrity Protection (SIP) is a security feature that protects all the Apple applications and utilities, certain system processes, files, and folders from being modified. Thus, the /System, /bin, /sbin, and /usr folders (except the /usr/local folder) are protected.

SIP is enabled by default but can be disabled from the OS X Recovery partition.

To disable SIP:

Quit OnyX.

Restart the computer while holding down Command-R until the OS X Utilities pane appears.

In the Utilities menu, select Terminal.

Enter the following command and press Return:

csrutil disable

Quit Terminal.

Restart normally from the Apple menu ().

To enable SIP:

Quit OnyX.

Restart the computer while holding down Command-R until the OS X Utilities pane appears.

In the Utilities menu, select Terminal.

Enter the following command and press Return:

csrutil enable

Quit Terminal.

Restart normally from the Apple menu ().
