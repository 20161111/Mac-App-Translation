Spotlight index

Spotlight, the intelligent search function of OS X, is very fast because it creates and uses an index of all files on all connected hard disks. This option rebuilds the index. A few seconds after deleting it, the system will automatically rerun the content indexing of the hard disks in the background. You can see the progress of this action by clicking on the Spotlight icon in the menu bar and then entering any word into the Spotlight Search box; the word "Indexing..." and an aqua progress indicator (a moving bar) will appear while indexing is occurring. If you restart your computer during this process, the indexing is stopped and will be resumed after restarting.

If Indexing is turned off, removing the index will automatically turn indexing back on.

The Spotlight index can be rebuilt from the Rebuilding pane of the Maintenance toolbar item or from the Automation pane.
