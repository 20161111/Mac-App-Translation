AppleDouble

You can use the AppleDouble pane of the Utilities toolbar item to delete the AppleDouble files. These files (whose names begin with "._") are created by the system when an application adds resource data or metadata to a file on a disk whose format doesnâ€™t natively support resources or metadata (such as MS-DOS, NFS, SMB, WebDAV, or UFS).

Warning: Some rare files will be made unusable if their ._ AppleDouble file is deleted (for example, EPS files).

Click the Select button to select a disk or a folder; then click OK.

This operation takes a long time on a high–capacity volume. Wait for the task to finish.
