locate database / Finding

locate is a Unix command that you can use to quickly find a file or a folder by searching the locate database (which is an index of all items stored on the hard disk). When installing the system, this database doesn't exist, and you must create it and update it regularly if you want to use the locate command.


The Finding pane of the Utilities toolbar item:

You can use the Finding pane of the Utilities toolbar item to create, remove, and update the locate database. After the database has been created, then you can search for items. Click a list item to have it appear in the Finder. This pane also displays the size of the database and the date of last update.

The Rebuilding pane of the Maintenance toolbar item:

If the locate database has already been created, it can be updated from the Rebuilding pane of the Maintenance toolbar item.

Warning: Creating and rebuilding the locate database can take a long time. Wait for the task to complete.
