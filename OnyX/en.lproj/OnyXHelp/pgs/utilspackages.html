Packages

All of the Apple applications and some third–party applications are installed from a package file (.pkg). When installed, a receipt file (.bom) is created and stored in a hidden system folder (/private/var/db/receipts/ or /System/Library/Receipts/).

You can use the Packages pane of the Utilities toolbar item to open receipt files to show all the items installed by a package. First, click the Open Receipt button, and select the receipt tools that you may find; the installed items will appear in the "Installed items list" of the Packages pane. Then click an item in the window to reveal that selected item in the Finder, or click the "Reveal in the Finder" button to show the chosen receipt file (.bom) in a window of the Finder.
