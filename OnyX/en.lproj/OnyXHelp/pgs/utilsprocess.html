Process

You can use the Process pane in the Utilities toolbar item to turn on the Process accounting option, which is turned off by default.

If this option is turned on, OS X will save all processes executed by all the users of the computer in a file (/private/var/account/acct). You can also display and delete the content of this file.

Important: Many processes are in use, so this file can become very large. You should delete its content occasionally. This operation is launched while deleting the logs using the Automation pane or using the Logs pane of the Cleaning toolbar item.
