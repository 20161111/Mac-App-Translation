Locking Clips
You can lock a clip so that it can’t be altered in any way. This is useful when you have a clip placed or configured exactly the way you want and do not want to alter them. Figure 31 depicts a clip which is locked—it displays dimmed and cannot be modified or moved.
To lock a clip, select it and use one of these methods:
•
Select Edit > Lock Clip.
•
Click the gear icon on the clip and select Show Clip Inspector, then check Locked or select Lock Clip (Option-Command-L).
Figure 31. Locking and Unlocking Clips
To unlock the clip, select it and select Edit > Unlock Clip or display the gear menu and select Show Clip Inspector or select Unlock Clip (Option-Command-L).
