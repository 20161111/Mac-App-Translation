Creating a Freeze Frame
If you want to pause on a single frame of video while editing, select the clip and move the scrubber to the frame where you want to freeze. Select Edit > Add Freeze Frame to create a still image of the current frame of video, as shown below in Figure 33.
Figure 33. Creating a Freeze Frame
This command splits the video into two clips, and adds a two second freeze frame clip (essentially an image) in between the two split clips. The duration of the freeze frame can be modified by expanding or contracting the clip (see Trimming Clips).
