Generating Closed Captions
You can generate closed captions when exporting or publishing in these formats:
•
Export the project as an MP4 (H.264 video) file. Select the Web-High or Web-Low preset or any Apple device preset, or a customized derivative. You’ll need to check Add Captions Track on the export window. For details, see Exporting Your Video.
•
Publish the project to YouTube. You’ll need to check Add Captions Track on the export window. For details, see Publishing to YouTube.
