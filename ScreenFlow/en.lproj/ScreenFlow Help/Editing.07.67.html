Adding an Annotation Clip
To add an annotation clip, set the scrubber in the location where you want the clip to start, and click Add Annotations or click the gear icon and select Add Annotations Clip or select Insert > Annotations Clip. ScreenFlow adds a 5 second annotation clip in a new, top track (which you can move anywhere on the timeline you want).
When you create an annotation clip, you must immediately add at least one object to the clip, or ScreenFlow will automatically delete it. As a matter of fact, if you later remove all annotations, ScreenFlow deletes the annotation clip automatically.
