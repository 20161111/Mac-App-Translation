Using the Countdown Window
By default, each time you start recording (unless youâ€™ve disabled it in Preferences), ScreenFlow displays the countdown window as shown below in Figure 17, providing a delay to allow you to prepare for recording.
Figure 17. Using the Countdown Window
By default, the countdown is set to 5 seconds. To change the countdown option (or to enable or disable the countdown window), select > Preferences > General. For details, see General Pane. You can set the countdown delay for up to ten seconds.
To skip a countdown in progress, simply click on the countdown window and ScreenFlow begins recording immediately.
