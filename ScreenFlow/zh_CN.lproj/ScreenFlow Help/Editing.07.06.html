Deleting All Unused Media
It is possible for media sources to exist in a ScreenFlow document without having an associated clip for them in the timeline. ScreenFlow provides an easy way to delete such orphaned media sources from the Media Library.
To do this, click the Media Library tab, click the gear icon to open its menu, then select Delete All Unused Media.
When the list of unused media displays, click Delete to delete all the unused sources from the Media Library.
