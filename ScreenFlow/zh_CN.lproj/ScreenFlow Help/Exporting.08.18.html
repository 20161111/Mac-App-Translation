Publishing Your ScreenFlow Project : Upload Manager Interface
Upload Manager Interface
Once the publishing settings have been selected and the Publish button is clicked, the exporting begins.
When the export has completed, the upload will occur in the background through the Upload Manager. The Upload Manager button displays in the main document window and displays progress via its blue progress bar.
Click the Upload Manager button to display details of the upload. (ScreenFlow supports multiple concurrent uploads.)
When the upload finishes, you can click the arrow icon to open the URL in a browser. Click Clear to close the Upload Manager and clear out completed uploads from the list. If there are no remaining uploads in progress, the Upload Manager button is hidden until a new upload occurs. (This is similar to how the Safari download manager works.)
If an error occurs when uploading, it is displayed in the Upload Manager window. Click the yellow warning triangle to display more information about the error.
