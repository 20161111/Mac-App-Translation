OS X Lion Save Commands

Table 33. Mac OS Lion File Menu Save Commands
Menu Item
Description
Save
(Command-S)
Saves the document with recent changes.
If the document has not been previously saved, ScreenFlow displays the Save As dialog, so that you can name the document and save it before closing it.
Duplicate
Saves a new version of the document without overwriting the previous version.
This allows you to save the ScreenFlow project with a new name and/or location.
