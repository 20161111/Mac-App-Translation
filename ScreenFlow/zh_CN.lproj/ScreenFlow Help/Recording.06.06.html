Configuring Recording in the Add Additional Recording Window
You can also use the Add Additional Recording window to select recording options, and record media to add to an open project document. To display the Add Additional Recording window in ScreenFlow, select File > Add Additional Recording (Figure 17) or click the Add Recording button in the media library, which is described in detail in Using the Media Library.
