Default Script Size and State

The appearance of a new empty script window is up to you. To set the appearance of new script windows:

  1. Create a new script window and set it up the way you want it. This might include such features as:

    • Size
    • View options
    • Configuration of panes and inspectors, including the event log

  2. Choose Window > Set Default Script Size & State.

From now on, new script windows will have these same characteristics.

Note: Aspects of the content (such as text) of a script are not part of the default script window appearance. To preserve script content for reuse in new scripts, make a user template.

To revert to Script Debugger’s default appearance for new windows:

  • Choose Window > Reset Default Script Size & State.
