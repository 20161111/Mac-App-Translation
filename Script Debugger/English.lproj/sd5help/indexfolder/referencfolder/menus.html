Menus

This is a reference section describing all of Script Debugger’s menus.

You can customize the keyboard shortcut for a menu item:

  • Open Script Debugger’s Preferences window and go to the Key Bindings pane.

Hint: In some cases where you find yourself using a command often, you may find that a toolbar item is supplied. Choose View > Customize Toolbar and examine the available toolbar items. If the command has a toolbar item, you can drag it into the toolbar, and from then on you can use that toolbar item as a way of issuing the command.
