View Options in a Script Window

Script Debugger provides many options for helping you view your script in a script window.

Besides choosing whether to wrap long lines, you can also show or hide invisible characters and spaces, see line numbers and tab stops, and even view your script’s raw Apple event codes.

Many of these options apply in other contexts as well, such as value explorers.
