Apple Event Codes

In place of the English-like terminology in the dictionary window’s info pane, Script Debugger can display the raw Apple event codes to which they correspond. This can be useful when you want to analyze a raw Apple event (as displayed in the event log pane, for instance), or when you need to track down a terminology clash.

To toggle the visibility of raw Apple event codes:

  • Choose View > Show Raw (Chevron) Syntax. If the menu item is checked, Apple event codes are showing.

  • Or, click the Raw Syntax button in the dictionary window’s toolbar so that it is highlighted (white icon on black background). The button is optional; if you don’t see it, you can add it by customizing the toolbar.

You can also see raw Apple event codes in scripts, in an outliner, and in the event log window.
