Tab Stops

Script Debugger can display tab stops in text. Tab stops are the horizontal points at which indentation automatically occurs when a script is compiled or when a value is pretty-printed. The positions of the tab stops are shown by vertical lines behind the text. To show tab stops:

  • Choose View > Show Tab Stops.

  • Or, choose Show Tab Stops from the contextual menu.

If the menu item is checked, tab stops are showing.

The size of tab stops (the number of spaces to which each tab corresponds) is determined by the Editor preference, “Tab Width.”
