Batch export multiple files

With batch export, you can modify multiple images by:

  • Converting the files to another format.
  • Saving a group of images to a new file location.
  • Applying a naming scheme to the files.

Batch export multiple files:

  1. Open Snagit Editor.
  2. Select File > Batch Export.
  3. Select the files to export. Command-click to select multiple files.
  4. Click the Export button.
  5. Select a location and file format for the files.
  6. If you to apply a naming scheme to all the files, select the Sequence File Name checkbox and enter a name.
  7. Click Export.
