Capture preferences

The following preference options are available when you select Snagit > Preferences > Capture tab:

  • Hide Snagit when capturing: Minimize Snagit during capture.
  • Preview in Editor: Open the capture in Snagit Editor to preview, enhance, or share the capture.
  • Always copy captures to clipboard: Automatically copy the capture to the Clipboard for pasting after capture. Press Command-V to paste the capture from the Clipboard into another application or document.
  • Capture cursor: Capture the cursor in image captures. Snagit captures the cursor as a vector object and it is editable in Snagit Editor for image captures. Snagit automatically captures the cursor in video captures.
  • Show Video Countdown: Displays a three second countdown before starting the video capture recording.
  • Delay capture by: Use the delay capture option for extra time to set up the screen for a capture or for capturing a menu. See Set a capture delay.
  • Scrolling capture is disabled: This option only appears when Snagit's scrolling capture is disabled. Click the Enable button to install additional options to improve Snagit's scrolling capture capabilities. See Fix scrolling capture.
  • System Audio: Install or uninstall the TechSmith Audio Capture Component. The Techsmith Audio Capture Component allows you to record system audio (the sound from your computer speakers) during a video capture. See Record system audio.
