Combine image captures

You can assemble multiple images together on the canvas. Save as a Snagit Project File (.snagproj) to later edit the size or position on the objects within the image.

Combine captures:

Drag a capture from the tray onto another capture on the canvas.

An alternative way to combine a capture:

  1. Click to select the capture to copy in the tray. Select Edit > Copy or press Command-C.
  2. Click to select the capture in the tray to paste the other capture into. Select Edit > Paste or press Command-V.

    You may want to flatten the objects in a capture before combining captures to make the objects a single layer. To flatten an object, click to select the object and select Edit > Flatten.
