Flatten objects on the canvas

Flatten an object on the canvas to save the current appearance and position of an object permanently in an image. Once flattened, the vector objects cannot be separated from the image.

To flatten an object on the canvas:

    • Select the object and select Edit > Flatten.
    • Control-click the object and select Flatten.
    • Or, select the object and press Command-Shift-T.

To flatten all objects on the canvas:

  • Select Edit > Flatten All.
  • Right-click on the Canvas and select Flatten All.
  • Or, press Control-Shift-T.
