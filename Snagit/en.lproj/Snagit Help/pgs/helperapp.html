Troubleshooting Snagit helper applications

To help Snagit run properly, you may need to manually quit some Snagit applications.

  1. Select Applications > Utilities > Activity Monitor.
  2. Type "Snagit" into the Filter field in the upper right corner of Activity Monitor.
  3. A list of any Snagit helper applications appears. Select an application from the list and click the Quit Process button.
  4. Click Force Quit.
  5. Repeat steps 3 and 4 for the remaining Snagit applications.
  6. Relaunch Snagit.
