Working with objects on the canvas

The canvas is the area for previewing and editing captures. You can edit vector-based objects directly on the canvas. If you want to edit the vector-based objects within a capture at a later time, save to the .snagproj file format.

Edit objects on the canvas:

To edit an object, you must first click to select the object on the canvas. Flattened objects can not be edited.

  • Move: Drag the object to a new position on the canvas.
  • Resize: Drag a handle in or out.
  • Flip: Select Arrange > Flip Horizontal or Flip Vertical.
  • Rotate: For callouts and pasted captures, drag the green handle around the object.

    To rotate an entire capture, select Edit > Rotate Image. Enter the rotation degree and click Apply.
  • Flatten: Control-click on the object and select Flatten.
