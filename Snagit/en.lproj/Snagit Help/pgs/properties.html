Adjust the Tools or Effects properties

Customize the settings for the selected tool or effect in the properties. To access the properties:

  • For Tools: Click to select a tool on the Tools pane. The properties appear below the styles gallery.
  • For Effects: Click the on the effect in the Effects pane.

To learn how to create your own style in the gallery based on the customized properties, see styles gallery.

Select the color:

  • Select color: Select the border/fill/shadow colors.
  • Switch colors: Switch the top and bottom colors.
  • Color picker: Click the eyedropper icon to select a color from the image.

See Select a color to learn how to select a custom color, HEX, RGB, or CMYK color value.

Opacity:

Set the transparency level. This property is available for the callout, arrow, stamp, pen, line, and shape tools and the border, shadow, and reflection effects.

Shadow:

The shadow property controls the appearance of the shadow on an object. Click an area in the shadow control to apply or remove a shadow.

The shadow effect provides additional shadow properties such as angle selection, color, opacity, and offset.

Font:

Click the Font button to select the font, typeface, size, and color for text only or text within callouts. You must select the text to apply the property changes.

Line/Size:

  • Line style: Select from solid and dotted lines for the borders of callouts or shapes or the line style in lines and arrows.
  • Line width: Determines the width of the line or border for the callout, shape, line, and arrow tools.
  • Size: Determines the width of the eraser or pen tools and the size of the edge, border, and reflection effects.

Shape:

Select a shape from the dropdown for callouts or shapes.
