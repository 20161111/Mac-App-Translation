Share to Screencast.com

Upload an image or video onto Screencast.com for hosting. Screencast.com provides a URL for quick sharing or an embed code for adding the capture onto a website or blog. Share to Screencast.com to:

  • Collaborate quickly on projects without slowing down to save or open attachments.
  • Avoid the concerns faced when sharing between two computers or over a network such as viruses, file sizes, and incompatibility.
  • Include visuals to communicate in places that don't accept attachments (e.g. blog comments, forum posts, twitter).
  • Provide in depth feedback without a long email write-up.
  • Allow comments on images or videos, control who accesses the content, offer an RSS feed, and more with Screencast.com's additional features.

Share to Screencast.com

  1. Click the Screencast.com output button or select Share > Screencast.com.
  2. The Screencast.com Sign In dialog appears if you have not saved your Screencast.com account settings in the Preferences. See Customize the Screencast.com account settings below.
    • If you have not previously shared to Screencast.com, select Create a new account.
    • If you have previously shared to Screencast.com, select Use my existing account and enter your email and password. Click Log In.

Snagit uploads the capture onto Screencast.com and copies the link or embed code is onto the Clipboard for pasting and sharing into an email, instant message, or another location.

Customize the Screencast.com account settings

You can customize, save, and set up multiple Screencast.com accounts from the Output Preferences dialog.

  1. Select the More Outputs button and click the Output Preferences button.
  2. Click the Screencast.com option. The Screencast.com output settings appear below.
  3. Click Sign In to set up an account username and password. Select Save Password in Keychain to remember this account. Click Log In.
  4. From the Clipboard Contents dropdown, select one of the following:
    • URL: Provides the Screencast.com URL after uploading to allow you to quickly paste the URL for sharing.
    • HTML Embed Code: Provides the embed code after uploading to embed the capture on a website or blog.
  5. Select the Folder to upload the capture to. Snagit is the default folder.
  6. To allow users to comment on the capture, enable the Allow commenting on Screencast.com option.
  7. Close the Preferences dialog.

Snagit remembers these Screencast.com account settings. To share a capture with these settings, click the Screencast.com output button above the tray.
