Add text

Tools and effects are only available for image captures.

Add text:

  1. Click to select the Callout tool on the Tools tab.
  2. Select the text option from the style gallery.
  3. Click the Font button in the properties to edit the font, color, typeface, and size.
  4. Click and drag to create a text box on the canvas.
  5. Double-click within the text box on the canvas and enter text.
