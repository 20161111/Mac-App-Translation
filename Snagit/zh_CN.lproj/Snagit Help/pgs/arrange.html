Arrange objects on the canvas

You can control the order objects appear on the canvas.

Change the order of objects

  1. On the Tools tab, click the Object Selector tool.
  2. Click to select the object to change on the canvas. Flattened objects cannot be selected.
  3. To change the order the object appears within the layers on the canvas, select Arrange > Move to Front, Move Forward, Send to Back, or Send Backward.
