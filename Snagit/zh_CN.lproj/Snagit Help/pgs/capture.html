Capture types and options

  • Global Capture (Control-Shift-C): Capture an image, video, window, region, scrolling capture, or the full screen all from a single capture type.
  • Image Capture (Control-Shift-S):
  • Video Capture (Control-Shift-V): Record the action on the screen as a MPEG-4 video file. Choose to include microphone or system audio.
  • Menu Capture (Control-Shift-E): Capture a dropdown, flyout, or cascading menu.
  • Webcam Capture: Capture a still image from a web camera.
  • Web Page Capture: Captures the entire web page. Use this capture type as an alternative method to scrolling capture.
  • Window Capture (Control-Shift-W): Capture an open or hidden window.
