Change or restore the capture keyboard shortcuts

The capture keyboard shortcuts allow you to quickly capture anything on your screen while maintaining the cursor's position. You can change the following default keyboard shortcuts:

  • All-in-One Image Capture (Shift-Control-C)
  • Window Capture (Shift-Control-W)
  • Menu Capture (Shift-Control-E)
  • Video Capture & Stop Recording (Shift-Control-V)
  • Start & Pause Recording (Option-Shift-Command-S)

For a complete list of the Snagit keyboard shortcuts, see keyboard shortcuts or view the Snagit menus.

Change a capture keyboard shortcut in the Capture window:

The Global (All-in-One) Capture, Image Capture, and Video Capture keyboard shortcuts can be changed in the Capture window.

  1. In the Capture window, click the Additional Capture Options button.
  2. Click in the keyboard field for the keyboard shortcut to change.
  3. Type the desired keyboard combination. Click the Undo button to reset to the previous keyboard shortcut.

    Keyboard shortcuts can be any 2-5 keys on the keyboard. Any keyboard shortcuts already assigned to the Mac operating system or to Snagit are not available.

Change a capture keyboard shortcut in the Preferences:

The Global (All-in-One) Capture, Image Capture, Video Capture, and Window Capture, and Menu Capture keyboard shortcuts can be changed in the Preferences.

  1. In the Capture window, click the Additional Capture Options button.
  2. Click the More Preferences buttton.
  3. On the Keyboard tab, click in the keyboard field for the keyboard shortcut to change.
  4. Type the desired keyboard combination. Click the Undo button to reset to the previous keyboard shortcut.

    Keyboard shortcuts can be any 2-5 keys on the keyboard. Any keyboard shortcuts already assigned to the Mac operating system or to Snagit are not available.

Restore the default keyboard shortcuts:

  1. Select Snagit > Preferences > Keyboard tab.
  2. Click the Restore Shortcuts button.
  3. Close the Preferences dialog.
