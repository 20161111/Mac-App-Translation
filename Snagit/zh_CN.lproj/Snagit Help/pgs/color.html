Select a color

You can select a custom color within the properties on the Tools or Effects tabs. You can enter a RGB value (commonly used for digital images), CMYK value (commonly used for four-color printing), or a web safe value. To learn more about the color properties, see properties.

Access the Color properties:

  1. To view the properties:
    • On the Tools tab, click to select a tool.
    • On the Effects tab, click the Properties icon on an effect.
  2. In the properties, click the color to change.
  3. Select other in the dropdown list. The Colors dialog box opens.

Select a RGB color:

  1. In the properties, click the color to change.
  2. Select other in the dropdown list. The Colors dialog box opens.
  3. Click the Color Sliders tab and select RGB Sliders from the dropdown list.
  4. Use the sliders or enter RGB values into the fields.

Select a web safe color:

  1. In the properties, click the color to change.
  2. Select other in the dropdown list. The Colors dialog box opens.
  3. Click the Color Palettes tab and select Web Safe Colors from the dropdown list.
  4. Select a web safe color from the list.

Select a CMYK color:

  1. In the properties, click the color to change.
  2. Select other in the dropdown list. The Colors dialog box opens.
  3. Click the Color Sliders tab and select CMYK Sliders from the dropdown list.
  4. Use the sliders or enter CMYK values into the fields.
