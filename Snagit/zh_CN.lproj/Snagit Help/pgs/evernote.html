Share to Evernote

The Evernote output uploads the selected image capture into a new note. To learn more about Evernote, visit Evernote's website.

Share to Evernote

  1. Select the image capture to share from the tray.
  2. Click the Evernote output button or select Share > Evernote.
  3. If you have not previously saved your user information, you will need to enter a username and password and click Log In.
    • If you selected Send to Evernote application in the Snagit Output preferences, Snagit inserts the capture into a new note. You can enter body text, enter a note title, and select the Notebook.
    • If you selected Upload to Evernote on the web in the Snagit Output preferences, the Send to Evernote dialog appears. Enter a title and body text and click Share. The capture is inserted into the body of a new note. The new note appears in the Notebook selected in the Snagit Output preferences.

Customize the Evernote account settings

You can customize, save, and set up multiple Evernote accounts from the Output Preferences dialog.

  1. Select the More Outputs button and click the Output Preferences button.
  2. Click the Evernote option. The Evernote output settings appear below.
  3. Click Authorize to set up an account username and password. Select Save Password in Keychain to remember this account. Click Sign in.
  4. Select to Send to Evernote application or Upload to Evernote on the web.
    • If you select Upload to Evernote on the web, select the Notebook to add your new note to.
  5. Close the Preferences dialog.

Snagit remembers these Evernote account settings. To share a capture with these settings, click the Evernote output button above the tray.
