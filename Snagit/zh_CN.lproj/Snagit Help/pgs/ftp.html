Share by FTP

The FTP output uploads the selected capture to your FTP server for publishing on a website.

Share capture by FTP

  1. Click the FTP output button or select Share > FTP.
  2. The FTP login dialog appears. See Customize FTP settings below if you have not previously shared by FTP.
  3. Click the Share button.

Customize your FTP settings

  1. Select the More Outputs button and click the Output Preferences button.
  2. Click the FTP option. The FTP output settings appear below.
  3. Enter the server address.
  4. Select either FTP or SFTP. Then enter the port number (usualy 21 for FTP and 22 for SFTP), directory (optional), and username and password to the account.
  5. Click the Test button to verify the settings.
  6. Select Custom Code from the Clipboard contents dropdown to obtain a link or specialized code for your content.
