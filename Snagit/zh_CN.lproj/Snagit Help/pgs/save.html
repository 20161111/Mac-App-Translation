Save a capture

Snagit automatically stores any unsaved captures as Snagit project files (.snagproj) in the Documents > Snagit > Autosaved Captures folder. Once a capture is saved, the autosaved .snagproj file is deleted from the folder. A appears on any saved capture in the tray with unsaved changes.

Save a capture:

  1. Select File > Save (Command-S) or File > Save As (Command-Shift-S).
  2. The Save As dialog appears. Enter a title.
  3. Select a file format and location.
  4. Click Save.
